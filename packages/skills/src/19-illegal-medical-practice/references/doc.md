# การสอบสวนคดีทำการประกอบโรคศิลปะ (เพื่อแสวงหาประโยชน์โดยไม่ได้ขึ้นทะเบียนและไม่ได้รับใบอนุญาต)

## การสอบสวนควรดำเนินการดังต่อไปนี้

1. ให้สอบสวนผู้กล่าวหาให้ได้รายละเอียดว่า ผู้กล่าวหาทราบได้อย่างไรว่า ผู้ต้องหาได้กระทำความผิดเกี่ยวกับโรคศิลปะโดยไม่ได้รับอนุญาต รู้ด้วยตนเองเกี่ยวกับการสืบสวน หรือมีคนมาบอก ถ้ามีคนมาบอกผู้นั้นเป็นใคร บอกว่าผู้ต้องหาเป็นใคร ทำอะไร ที่ไหน เมื่อไหร่ และทำอย่างไร ที่เห็นว่าได้กระทำผิดกฎหมาย ถ้าทราบเองโดยการสืบสวนไปพบการกระทำผิดด้วยตนเองหรืออย่างไร ถ้าพบด้วยตนเองผู้ต้องหาทำอะไรกับใคร และทำอย่างไร ได้สอบถามผู้ต้องหาหรือไม่ว่า ผู้ต้องหาประกอบโรคศิลปะมีทะเบียนหรือใบอนุญาตให้ประกอบโรคศิลปะสาขาใด เมื่อใด เหตุเกิดที่ไหน วัน เดือน ปี เวลาใด
2. ให้สอบสวนผู้ต้องหา ถ้าผู้ต้องหาให้การรับสารภาพ ต้องสอบสวนให้ได้รายละเอียดว่า ผู้ต้องหาได้รับการศึกษาระดับใด เคยทำงานเกี่ยวกับการรักษาโรคของคนป่วยมาหรือไม่ รู้วิธีการซื้อหรือสั่งยาให้กับผู้ป่วยหรือไม่ การใช้เครื่องอุปกรณ์การแพทย์แต่ละอย่างใช้อย่างไร มีทะเบียนและใบอนุญาตให้ประกอบโรคศิลปะหรือไม่ ถ้ามีใบอนุญาตถูกเพิกถอน สั่งพักจากทางราชการหรือไม่ ผู้ต้องหาได้ไปรักษาคนป่วยชื่ออะไร อยู่ที่ไหน เป็นโรคอะไร ผู้ต้องหาให้ยาคนป่วยด้วยวิธีใด ได้รับผลประโยชน์ตอบแทนอย่างไรบ้าง ขณะที่ผู้ต้องหาถูกจับกุม ผู้ต้องหาทำอะไร กับใคร ที่ไหน และทำอย่างไร เคยต้องหาต้องโทษคดีอาญามาก่อนหรือไม่
3. ให้สอบสวนพยานให้ได้รายละเอียดว่า พยานรู้จักกับผู้ต้องหา เคยมีสาเหตุโกรธเคืองกันมาก่อนหรือไม่ ได้พบผู้ต้องหาทำอะไรกับใคร ที่ไหน และทำอย่างไร ทราบหรือไม่ว่าผู้ต้องหาเป็นหมอหรือแพทย์ที่แท้จริง ถ้าพยานถูกผู้ต้องหารักษาโรคให้ พยานเป็นโรคอะไร รู้จักกับผู้ต้องหาเมื่อไร รู้จากใครบอกหรือใครแนะนำ ผู้ต้องหารักษาโรคให้พยานโดยวิธีใดบ้าง และพยานเสียเงินค่ารักษาให้กับผู้ต้องหาเท่าไร มีใครรู้เห็นบ้าง
4. ถ้าการยึดเครื่องมือเครื่องใช้ ยารักษาโรค ได้จากผู้ต้องหา ต้องส่งให้แพทย์สาธารณสุขอำเภอหรือแพทย์สาธารณสุขจังหวัดตรวจ หรือส่งกองพิสูจน์หลักฐานกรมตำรวจหรือกระวิทยาศาสตร์การแพทย์ ทำการตรวจว่า เครื่องมือเครื่องใช้เป็นเครื่องมือสำหรับแพทย์หรือพยาบาลใช้ ใช่หรือไม่ ยาต่างๆ ที่ยึดได้เป็นยาชนิดใด ใช้ในการรักษาโรมนุษย์ได้หรือไม่ และถ้าใช้ไม่ถูกต้องจะเป็นอันตรายแก่ชีวิตและร่างกายหรือไม่
