# การสอบสวนคดีกรรโชก

## 1. สอบสวนผู้กล่าวหาหรือผู้เสียหายให้ได้ความตามองค์ประกอบความผิดฐานกรรโชก

1. มีการบังคับโดย
   - ใช้กำลังประทุษร้าย
   - หรือขู่เข็ญขืนใจ
2. ให้ผู้เสียหายสัญญาว่าจะ
   - ส่งมอบทรัพย์
   - หรือให้ทำ / ถอน / ทำลายหนังสือสำคัญ
3. ผู้เสียหายยอมตามการบังคับนั้น
4. ผู้ต้องหามีเจตนาทุจริต

## 2. ขณะถูกกรรโชก ผู้เสียหายทำอะไร อยู่ที่ไหน กับใคร เมื่อใด

(วัน–เวลา–สถานที่เกิดเหตุ)

## 3. รายละเอียดเกี่ยวกับผู้ต้องหา

- จำนวนกี่คน
- เป็นหญิงหรือชาย
- เคยรู้จักกันมาก่อนหรือไม่
- อายุประมาณเท่าใด
- แต่งกายอย่างไร
- รูปร่าง ลักษณะ ตำหนิรูปพรรณ
- สำเนียงพูดเป็นคนภาคใด หรือเป็นคนต่างชาติ
- มีอาวุธหรือไม่ เป็นอาวุธชนิดใด

## 4. การกระทำของผู้ต้องหา

- ทำอะไรกับผู้กล่าวหาหรือผู้เสียหาย
- พูดจาบังคับหรือขู่เข็ญว่าอย่างไร
- สัญญาอะไร
- พาผู้เสียหายไปที่ใดบ้าง

## 5. ผู้กล่าวหาหรือผู้เสียหายตกลงทำตามสัญญาหรือไม่

- เสียหายอย่างไร

## 6. สอบสวนพยานที่รู้เห็นเหตุการณ์ทุกคนอย่างละเอียด

รวมถึงเจ้าหน้าที่ตำรวจผู้จับกุม (ถ้ามี)

## 7. พนักงานสอบสวนต้องตรวจสถานที่เกิดเหตุ

- บันทึกการตรวจสถานที่เกิดเหตุ
- ทำแผนที่สังเขป
- ถ่ายรูปประกอบ

## 8. หากจับผู้ต้องหาได้และรับสารภาพ ต้องสอบสวนละเอียดเกี่ยวกับ

- วิธีการกระทำผิด
- ผู้ร่วมกระทำผิดแต่ละคนทำอะไร
- แบ่งงานกันอย่างไร
- ทรัพย์สินที่ได้ไปมีอะไรบ้าง
- แบ่งกันอย่างไร
- อยู่ที่ไหน ต้องติดตามเอาคืนมาเป็นพยานหลักฐาน

## 9. นำผู้ต้องหาไปชี้ที่เกิดเหตุ

- ทำแผนประทุษกรรมก่อนเกิดเหตุ
- ขณะเกิดเหตุ
- หลังเกิดเหตุ
- แล้วสอบสวนเพิ่มเติมไว้
