# การสอบสวนความผิด เป็นเจ้าพนักงานให้ผู้ถูกคุมขังหลุดพ้นไป

ความผิดลักษณะนี้เกี่ยวข้องกับเจ้าพนักงานผู้มีหน้าที่ควบคุมผู้ต้องหา จำเลย หรือผู้ต้องขัง การสอบสวนต้องมุ่งพิสูจน์ว่าเจ้าพนักงานมีส่วนทำให้ผู้ถูกคุมขังหลุดพ้นจากการควบคุมโดยเจตนา ประมาท หรือเป็นเหตุสุดวิสัย

## 1. การสอบสวนผู้กล่าวหา

1. ตรวจสอบสถานะของผู้ต้องหา
   - ผู้ต้องหาเป็นเจ้าพนักงานหรือไม่
   - มีตำแหน่งหน้าที่ควบคุมผู้ต้องขังตามอำนาจของศาล พนักงานสอบสวน หรือเจ้าพนักงานผู้มีอำนาจสืบสวนคดีอาญาหรือไม่
2. การกระทำที่ทำให้ผู้ถูกคุมขังหลบหนี
   - ผู้ต้องหากระทำสิ่งใดที่ทำให้ผู้ถูกคุมขังหลุดพ้นจากการควบคุม
3. การนำผู้ถูกคุมขังออกจากสถานที่ควบคุม
   - นำออกจากเรือนจำ ห้องควบคุม หรือห้องคุมขังอย่างไร
   - ต้องมีเอกสารประกอบ เช่น ใบเบิกตัว คำสั่ง หรือเอกสารอ้างอิงอื่น ๆ
4. การใช้เครื่องพันธนาการ
   - ผู้ถูกคุมขังต้องมีเครื่องพันธนาการหรือไม่
   - มีการปฏิบัติตามระเบียบหรือไม่
5. เจตนาของผู้ต้องหา
   - ผู้ต้องหามีเจตนาปล่อยผู้ถูกคุมขังหรือไม่
   - หรือเป็นเพียงความประมาท
   - หรือเป็นเหตุสุดวิสัยที่ไม่สามารถป้องกันได้

## 2. การสอบสวนผู้ต้องหา (กรณียอมให้การ)

1. สถานะการรับราชการ
   - ตำแหน่ง
   - ประเภทข้าราชการ (สามัญ/วิสามัญ/ลูกจ้าง)
2. หน้าที่ควบคุมผู้ต้องขัง
   - มีอำนาจควบคุมผู้ต้องขังตามกฎหมายหรือไม่
3. สถานะของผู้ถูกคุมขังที่หลบหนี
   - เป็นนักโทษ จำเลย หรือผู้ต้องหา
   - อยู่ในอำนาจของศาล พนักงานสอบสวน หรือเจ้าพนักงานใด
4. สถานที่คุมขัง
   - ผู้ถูกคุมขังอยู่ที่ใด
   - อยู่ในอำนาจของหน่วยงานใด
5. อำนาจในการนำผู้ถูกคุมขังออกนอกสถานที่
   - มีอำนาจหรือไม่
   - อ้างอิงระเบียบ ข้อบังคับ หรือคำสั่งใด
6. การนำผู้ถูกคุมขังไปทำกิจกรรมนอกเหนือภารกิจ
   - เช่น ไปดื่มสุรา หรือไปทำงานอื่น
   - กระทำได้หรือไม่ ตามระเบียบใด
7. การใช้เครื่องพันธนาการ
   - ผู้ถูกคุมขังมีเครื่องพันธนาการหรือไม่ในขณะนำออกไป
8. วิธีการหลบหนี
   - ผู้ถูกคุมขังหลบหนีได้อย่างไร
   - ผู้ต้องหามีส่วนเกี่ยวข้องหรือไม่

## 3. การสอบสวนพยาน

1. การเบิกตัวผู้ถูกคุมขัง
   - ผู้ต้องหาไปเบิกตัวกับใคร
   - เบิกตัวที่ไหน
   - ขั้นตอนถูกต้องหรือไม่
2. การตรวจสอบสมุดเบิกตัว
   - ผู้คุมบัญชีหรือผู้รับผิดชอบสมุดเบิกตัวเกี่ยวข้องกับผู้ต้องหาอย่างไร
   - มีการทักท้วงหรือไม่
   - ผู้ต้องหาฝ่าฝืนหรือกล่าวอย่างไร
3. ความเป็นมาของผู้ถูกคุมขังที่หลบหนี สอบสวนให้ได้ว่า
   1. ผู้ถูกคุมขังต้องโทษในคดีใด
   2. ใครเป็นผู้จับกุม
   3. ใครเป็นพนักงานสอบสวน และได้ดำเนินการสอบสวนอย่างไร
