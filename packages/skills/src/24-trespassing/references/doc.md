# การสอบสวนคดีความผิดฐานบุกรุก

## ควรแยกการพิจารณาเกี่ยวกับการสอบสวน 2 อย่างคือ

### 1. บุกรุกเข้าไปในอสังหาริมทรัพย์ของผู้อื่น ควรสอบสวนให้ได้ความว่า

1. อสังหาริมทรัพย์นั้นกรรมสิทธิ์เป็นของใคร หรือใครเป็นเจ้าของ และใครเป็นผู้ครอบครองโดยชอบด้วยกฎหมาย
2. มีหลักฐานการเป็นเจ้าของหรือผู้ครอบครองอย่างไรบ้าง
3. ลักษณะของอสังหาริมทรัพย์นั้น จำเป็นต้องมีแผนที่ประกอบทั้งหมดหรือแต่บางส่วนที่ผู้ต้องหาบุกรุก
4. ผู้ต้องหาหรือกระทำผิดต้อง
   1. มีเจตนาที่จะมิให้เจ้าของหรือผู้ครอบครองอสังหาริมทรัพย์โดยปกติสุขและได้เข้าครอบครองอสังหาริมทรัพย์นั้น
   2. มีเจตนาเพื่อที่จะครอบครองทรัพย์นั้นโดยการยักย้าย หรือทำลายสิ่งต่าง ๆ ให้ผิดไปจากเดิม ได้เข้าไปภายในเขตอสังหาริมทรัพย์ของเขาโดยมีเหตุอันสมควรหรือไม่
   3. มีอาวุธติดตัวไปหรือไม่ ใช้กำลังประทุษร้ายหรือขู่เข็ญว่าจะใช้กำลังประทุษร้ายหรือไม่
   4. กระทำด้วยกันกี่คน เวลากลางวันหรือกลางคืน

### 2. บุกรุกเข้าไปในเคหสถานของผู้อื่น ควรสอบสวนให้ได้ความว่า

1. สถานที่เกิดเหตุใช้เป็นที่อยู่อาศัย เช่น เรือน, โรง, เรือ, หรือแพ เป็นบริเวณที่ซึ่งใช้เป็นที่อยู่อาศัยนั้นด้วยตาม ป.อาญา มาตรา 1(4) หรือไม่
2. ที่อยู่อาศัยมีรั้วล้อมหรือไม่ ถ้ามีลักษณะแข็งแรง ทนทาน ต่ำสูง สร้างด้วยอิฐ คอนกรีต ไม้ หรืออะไร
3. ก่อนสอบสวน พนักงานสอบสวนจะต้องไปดูแลและตรวจสถานที่เกิดเหตุ ถ่ายรูป ทำแผนที่สังเขปก่อน จะได้สะดวกเกี่ยวกับการสอบสวนผู้ต้องหา (ถ้ามี) และผู้เสียหาย ผู้กล่าวหา และพยาน
4. เคหสถานนั้นใครเป็นเจ้าของ หรือใครเป็นผู้ครอบครองโดยชอบด้วยกฎหมาย

## การสอบสวนผู้เสียหายให้ได้ความว่า

1. เคหสถานนั้นใครเป็นเจ้าของ ถ้าเป็นที่พักอาศัยเป็นเรือนไม้ ตึกกี่ชั้น มีกี่ห้อง อยู่ด้วยกันกี่คน ใช้แสงสว่างด้วยอะไร
2. วัน เดือน ปี เวลา ที่ถูกบุกรุก และผู้บุกรุกมีกี่คน เข้าไปทางไหน ออกทางไหน ทำอะไรกับใคร ทำอย่างไร มีทรัพย์สินอะไรเสียหายบ้าง เห็นมีอาวุธหรือไม่
3. ทราบหรือไม่ว่า ผู้บุกรุกเป็นใคร รูปร่างลักษณะ ตำหนิรูปพรรณ ที่อยู่ของผู้บุกรุก รู้จักหรือเคยเห็นหน้ามาก่อนหรือไม่
4. มีพยานหลักฐานอย่างไรบ้าง อาจเป็นพยานบุคคล พยานวัตถุ พยานเอกสารหรืออื่น ๆ ที่ผู้บุกรุกทำให้เกิดขึ้น หรือทิ้งไว้บริเวณที่เกิดเหตุ
5. ทราบหรือไม่ว่าผู้บุกรุก (ผู้ต้องหา) มีเจตนาอย่างไร เคยมีสาเหตุโกรธเคืองกันมาก่อนหรือไม่
6. สถานที่เกิดเหตุ ตำบล อำเภอ จังหวัดใด
7. เอกสารใด ๆ ที่พนักงานสอบสวนจัดทำขึ้น ถ้าเกี่ยวข้องกับผู้เสียหาย หรือผู้กล่าวหา ควรนำให้ผู้เสียหายหรือผู้กล่าวหาตลอดจนพยานดู เช่น รูปถ่าย แผนที่สังเขป สถานที่เกิดเหตุ ถ้ารับว่าถูกต้องให้ลงลายมือชื่อรับรองไว้ด้วย
8. สิ่งอื่น ๆ ที่พนักงานสอบสวนเห็นว่า ควรสอบสวน

## การสอบสวนผู้ต้องหา ถ้ารับสารภาพควรสอบสวนให้ได้ความว่า

1. บุกรุกเข้าไปที่ไหน เมื่อใด เพื่อประสงค์อะไร
2. ไปกับใครบ้าง มีอาวุธอะไรติดตัวเข้าไปบ้าง
3. เข้าทางไหน ออกทางไหน
4. ไปทำอะไรกับใคร และทำอย่างไร
5. ทิ้งของส่วนตัวอะไรไว้ ณ ที่เกิดเหตุบ้าง
6. เคยต้องหาต้องโทษในคดีอาญาถึงจำคุกมาก่อนหรือไม่
7. และอื่น ๆ ที่ควรสอบถาม

## การสอบสวนพยาน ควรสอบสวนให้ได้ความว่า

1. รู้เห็นการกระทำของผู้ต้องหาตอนไหน (ก่อนเกิดเหตุ ขณะเกิดเหตุ หลังเกิดเหตุ) วัน เดือน ปี เวลาใด
2. เห็นผู้ต้องหากี่คน รู้จักกันมาก่อนหรือไม่ ตำหนิรูปพรรณอย่างไร แต่งกายอย่างไร เห็นมีอาวุธหรือไม่
3. รู้จักกับผู้ต้องหามาก่อนและมีสาเหตุโกรธเคืองกันหรือไม่
4. ถ้าเป็นเวลากลางคืนเห็นผู้ต้องหาโดยอาศัยแสงอะไรจากที่ไหน ระยะห่างกันประมาณเท่าไร
5. และอื่น ๆ ที่ควรสอบถาม
