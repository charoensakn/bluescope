# การสอบสวนคดีฆ่าช้างป่าหรือช้างเถื่อนในเขตอุทยานแห่งชาติ

1. การสอบสวนผู้กล่าวหาหรือพยาน ควรสอบสวนให้ได้ความว่า
   1. ช้างที่ถูกยิงหรือถูกล่า หรือถูกฆ่า เป็นช้างป่าหรือช้างเถื่อนหรือไม่ อายุช้างประมาณกี่ปี เป็นเพศผู้ หรือเพศเมีย ถ้าเป็นสัตว์ชนิดอื่นเป็นสัตว์ป่าสงวนหรือสัตว์ป่าคุ้มครองตาม พ.ร.บ. สงวนและคุ้มครองสัตว์ป่าหรือไม่
   2. บริเวณที่สัตว์ถูกล่าหรือถูกฆ่าอยู่ในเขตอุทยานแห่งชาติหรือไม่ ถ้าอยู่ในเขตอุทยานแห่งชาติ เป็นอุทยานแห่งชาติอะไร ตามพระราชกฤษฎีกากำหนดให้เป็นอุทยานแห่งชาติฉบับใด ประกาศใช้เมื่อไร
   3. แนวเขตอุทยานแห่งชาตินั้นอยู่ในท้องที่ตำบล อำเภอ จังหวัดใด
   4. สาเหตุที่สัตว์ที่ถูกล่าหรือถูกฆ่านั้น ถูกทำร้ายด้วยอะไร
   5. ผู้กล่าวหาหรือพยานที่ยืนยันชนิดของสัตว์ที่ถูกล่าหรือถูกฆ่าควรเป็นพนักงานเจ้าหน้าที่ป่าไม้ที่ได้รับการแต่งตั้งตามประกาศรัฐมนตรี หรืออาจเป็นผู้ที่มีความรู้เกี่ยวกับสัตว์ เช่น สัตวแพทย์อำเภอ สัตวแพทย์จังหวัด ฯลฯ เป็นต้น
   6. บริเวณอุทยานแห่งชาติอันเป็นสถานที่เกิดเหตุ พนักงานเจ้าหน้าที่ได้เคยอนุญาตให้ผู้ใดทำการล่าสัตว์บ้างหรือไม่
2. การสอบสวนผู้ต้องหา ควรสอบสวนให้ได้ความว่า
   1. ทราบหรือไม่ว่าบริเวณล่าสัตว์หรือฆ่าสัตว์อยู่ในเขตอุทยานแห่งชาติ ถ้าทราบ ทราบได้อย่างไร มีอะไรเป็นเครื่องหมายบอกไว้ เช่น แผ่นป้าย หลักเขต แนวเขต
   2. ใช้อะไรล่าสัตว์หรือฆ่าสัตว์ เช่น กับ ขวาก ตัง ตาข่ายหรือแห จั่นห้าว ปืนกล หรือปืนยิงเร็ว หลุมพราง ศรหรือหน้าไม้ที่ใช้ลูกอาบยาพิษ ยาพิษ วัตถุระเบิด ดอกไม้เพลิง ประทัด ยาเบื่อเมา ถ้าเป็นอาวุธปืนอย่างอื่น เช่น ปืนพก ปืนลูกซองยาว, สั้น ปืนแก๊ป ปืนเพลิง เป็นอาวุธปืนที่มีใบอนุญาตให้มี และใช้ตาม พ.ร.บ.อาวุธปืนฯ หรือไม่
   3. ใช้ยานพาหนะอะไร
   4. ล่า ยิง หรือฆ่า นำไปทำอะไร
   5. ไปล่าหรือยิงกี่คน ใครบ้าง
   6. ได้รับอนุญาตจากพนักงาน เจ้าหน้าที่หรือไม่
   7. และอื่น ๆ ที่ควรสอบสวน
3. ถ้าผู้ต้องหาใช้อาวุธปืนล่าหรือยิง ตรวจพบกระสุนปืน ปลอก กระสุนปืน หัวกระสุนปืน ควรส่งไปตรวจพิสูจน์ยังกองพิสูจน์หลักฐานว่า อาวุธปืนที่จับได้ ใช้ยิ่งได้ และสามารถประหัตประหารทำให้เกิดอันตรายแก่สัตว์ถึงบาดเจ็บหรือตายได้หรือไม่ ปลอกกระสุน หัวกระสุนปืน ใช้ยิงมาจากอาวุธปืนของกลางตามที่ยึดได้ ใช่หรือไม่
4. บริเวณสถานที่เกิดเหตุอยู่ในเขตอุทยานแห่งชาติจริงหรือไม่ ให้ทำแผนที่สังเขปประกอบสำนวนการสอบสวนไว้ เช่นเดียวกับการสอบสวนคดีความผิดฐานบุกรุก
5. ถ้าผู้ต้องหารับสารภาพ ควรนำผู้ต้องหาไปชี้ที่เกิดเหตุประกอบคำรับสารภาพด้วย แล้วทำการสอบสวนเพิ่มเติมไปตามลักษณะของคดี
6. ของกลางที่ยึดได้ประกอบคดี การนำผู้ต้องหาไปชี้ที่เกิดเหตุทำแผนประทุษกรรมควรถ่ายรูปไว้เป็นหลักฐาน แล้วบันทึกไว้ต่อหน้าพยาน
