# การสอบสวนคดีจำหน่ายยาเสพติด

## ประเด็นการสอบสวนผู้กล่าวหาและพยานจับกุม

1. ก่อนการจับกุม มีการสืบสวนทราบว่าผู้ต้องหามีพฤติการณ์จำหน่ายยาบ้าเมื่อใด ได้รายงานเป็นลายลักษณ์อักษรให้ผู้บังคับบัญชาทราบหรือไม่ อย่างไร หากรับแจ้งจากสายลับ เมื่อใด เหตุใดไม่ขอหมายค้นจากศาลเตรียมไว้ (กรณีที่ต้องค้นเพื่อพบพยานหลักฐานเพิ่มเติม)
2. การวางแผนเข้าล่อซื้อยาเสพติด
   1. หากมีการใช้สายลับเข้าล่อซื้อ ให้ปรากฏว่าสายลับมีกี่คน เป็นหญิงหรือชาย เท่าไร อายุ รูปร่าง การแต่งกายเป็นอย่างไร เหตุใดจึงนำมาสอบเป็นพยานไม่ได้
   2. วางแผนกันที่ไหน เวลาใด สายลับเข้าร่วมรับทราบการวางแผนด้วยหรือไม่
3. สายลับใช้โทรศัพท์ใคร (สั่งซื้อยาบ้าจากผู้ต้องหา) หมายเลขใด
   - หากใช้โทรศัพท์สาธารณะ ให้สอบถามสำนักบริการโทรศัพท์ว่าตู้โทรศัพท์หมายเลขใด มีการโทรไปยังโทรศัพท์ของผู้ต้องหาหรือไม่ อย่างไร
   - หลังจากจับกุมผู้ต้องหาแล้ว ให้ตรวจสอบโทรศัพท์มือถือของกลาง (ยึดจากผู้ต้องหา) ว่าพบหมายเลขโทรศัพท์ที่สายลับใช้ติดต่อหรือไม่
   - ต้องบันทึกรายละเอียด หมายเลขโทรศัพท์ วัน เดือน ปี เวลา ที่โทรติดต่อ แนบเป็นพยานเอกสาร
4. ตำรวจผู้จับกุมเดินทางไปจุดซุ่มด้วยยานพาหนะใด ของใคร
   - ระบุรายละเอียดเกี่ยวกับยานพาหนะโดยละเอียด
   - ขณะนั่งรอในรถ (ระหว่างสายลับล่อซื้อ) ให้ระบุจุดที่ผู้กล่าวหาและพยานนั่ง
   - สายลับเดินทางไปที่เกิดเหตุด้วยยานพาหนะใด ของใคร
5. จุดซุ่มอยู่ห่างจากที่เกิดเหตุเท่าใด
   - มองเห็นสายลับตลอดหรือไม่
   - แสงสว่างมาจากอะไร เช่น ดวงอาทิตย์ ดวงจันทร์ หลอดไฟ
   - ขนาดกี่วัตต์ ตั้งอยู่ตำแหน่งใด (ต้องระบุในแผนที่)
   - สภาพพื้นที่และการสัญจรของประชาชน
6. หลังสายลับล่อซื้อยาเสพติดแล้ว
   - นำยาไปให้ใคร ที่จุดใด
   - แจ้งรูปพรรณผู้ต้องหาหรือไม่
   - เมื่อจับกุมแล้ว ลักษณะตรงกับที่แจ้งหรือไม่
7. ขณะตรวจค้นจับกุม
   - มีบุคคลอื่นอยู่ในบ้านหรือไม่
   - เป็นใคร
   - มีประชาชนมามุงดูหรือไม่
8. ลักษณะของยาบ้า
   - สี
   - สัญลักษณ์
   - วิธีห่อหุ้ม
9. เหตุใดไม่บันทึกการจับกุม ณ ที่เกิดเหตุ ได้แจ้งสิทธิผู้ต้องหาหรือไม่

## ประเด็นการสอบสวนผู้ต้องหา

1. ประวัติส่วนตัว
   - การศึกษา
   - ความสามารถในการอ่านเขียนภาษาไทย
   - ทรัพย์สินของตนและครอบครัว
2. ชั้นจับกุม
   - ให้การอย่างไร
   - อ่านบันทึกเองหรือไม่
   - ลงชื่อหรือไม่
3. ชั้นสอบสวน
   - เข้าใจข้อหาหรือไม่
   - ยืนยันรับสารภาพหรือไม่
   - เหตุผลที่กระทำผิด
4. การขยายผล
   - ซื้อยาบ้ามาจากใคร
   - ชื่อ ที่อยู่ ลักษณะบุคคล
   - วิธีติดต่อซื้อขาย
5. จุดที่มีการซื้อขายยาเสพติด ระบุให้ชัดเจน (ถ่ายภาพประกอบ)
6. โทรศัพท์มือถือของกลาง
   - เป็นของใคร
   - ใช้ติดต่อซื้อขายอย่างไร
   - มีหมายเลขใดบันทึกไว้
7. รถจักรยานยนต์ของกลาง
   - เป็นของใคร
   - ใช้ในการกระทำผิดหรือไม่
   - ได้มาอย่างไร
   - หากไม่ใช่ของผู้ต้องหา → ต้องสอบเจ้าของ

## ประเด็นภาพถ่ายที่เกิดเหตุ

- ถ่ายภาพจากจุดซุ่มไปยังที่เกิดเหตุ
- เพื่อแสดงมุมมองของผู้จับกุม

## ประเด็นเกี่ยวกับของกลาง

1. ยาบ้าที่เหลือจากการตรวจพิสูจน์ ให้มีความเห็นริบ ตาม ม.102 พ.ร.บ.ยาเสพติดให้โทษ พ.ศ. 2522
2. โทรศัพท์มือถือของกลาง
   - หากใช้ในการซื้อขายยา → ริบ ตาม ม.30 พ.ร.บ.มาตรการปราบปรามยาเสพติด พ.ศ. 2534
   - หากซื้อจากเงินค้ายา → ดำเนินการตามข้อ 3
3. รถจักรยานยนต์
   - หากได้มาจากเงินค้ายา → แจ้ง ป.ป.ส.
   - เพื่อยึด/อายัดทรัพย์ ตาม ม.27 พ.ร.บ.มาตรการปราบปรามยาเสพติด พ.ศ. 2534
4. เงินสดที่ยึดได้ (ไม่ใช่เงินล่อซื้อ) หากมาจากการขายยาเสพติด แจ้ง ป.ป.ส. เพื่อยึด/อายัด ตาม ม.27 พ.ร.บ.มาตรการปราบปรามยาเสพติด พ.ศ. 2534
