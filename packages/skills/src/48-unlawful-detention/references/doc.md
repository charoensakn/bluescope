# การสอบสวนคดีหน่วงเหนี่ยวกักขัง ทำให้ผู้อื่นปราศจากเสรีภาพในร่างกาย

## 1. สอบสวนผู้เสียหายหรือผู้กล่าวหา ควรสอบสวนให้ได้ความว่า

1. วัน เดือน ปี เวลา และสถานที่เกิดเหตุ
2. ทำอะไรอยู่กับใครบ้าง
3. เคยรู้จักกับผู้ต้องหา หรือเคยเห็นหน้ามาก่อนหรือไม่
4. ผู้ต้องหากระทำกับตนอย่างไรบ้าง
5. ขณะที่ผู้ต้องหากระทำกับตนนั้น ผู้ต้องหาพูดอะไรบ้างหรือไม่ ถ้าพูด พูดว่าอย่างไร
6. ผู้ต้องหามีอาวุธอะไรติดตัวมาบ้าง
7. ได้รับอันตรายหรือบาดเจ็บที่ไหนบ้าง เช่น ฟกช้ำ ดำเขียว มีบาดแผล โลหิตไหล โลหิตซับ อันเกิดจากการกระทำของผู้ต้องหาที่กระทำด้วยกำลังกาย หรืออาวุธ ไปรักษาตัวที่ไหน รักษากี่วัน หาย ทำงานตามปกติได้ภายในกี่วัน
8. ลักษณะการกระทำของผู้ต้องหา ทำอย่างไรบ้าง มีการบังคับขู่เข็ญหรือไม่ เหมือนการกระทำของคนธรรมดา หรือคล้ายคนวิกลจริตหรือไม่

## 2. การสอบสวนผู้ต้องหา ถ้ารับสารภาพ ควรสอบสวนให้ได้ความว่า

1. มีอาชีพอะไร อยู่ที่ไหน ได้รับการศึกษาระดับใด
2. ได้กระทำอะไรกับใคร เมื่อไร ที่ไหน
3. อาวุธที่ใช้มีอะไร ได้มาจากไหน
4. สาเหตุที่กระทำ มีเจตนาประสงค์อะไร เช่น ให้ผู้เสียหายหรือผู้กล่าวหาช่วยเหลืออะไรบ้าง หรือเพื่อให้ญาติพี่น้อง บิดามารดาช่วยเหลือนำทรัพย์สินมาให้ หรือให้ผู้อื่น เช่น เจ้าหน้าที่ตำรวจจัดการตามวัตถุประสงค์ที่ได้บอกไป หรืออื่น ๆ
5. นอกจากผู้ต้องหาแล้ว มีผู้ใดร่วมหรือช่วยเหลือผู้ต้องหาอีกหรือไม่
