# การสอบสวนคดีลักทรัพย์ในเคหสถาน

ควรดำเนินการดังนี้

1. เมื่อมีผู้มาร้องทุกข์กล่าวโทษหรือมาแจ้งความ ร้อยเวรต้องสอบถามรายละเอียดเสียก่อน แล้วลง ปจว. รับแจ้งไว้พร้อมกับสอบสวนผู้กล่าวหาหรือผู้เสียหายให้ได้ความว่า
   - ทรัพย์ที่ถูกลักหรือถูกประทุษร้ายไปนั้นคืออะไร
   - ตำหนิรูปพรรณแต่ละอย่างเป็นอย่างไร ราคาเท่าไร
   - เก็บรักษาไว้ที่ไหนของบ้าน
   - เห็นทรัพย์ครั้งสุดท้ายเมื่อใด
   - รู้ว่าทรัพย์หายไป วัน เวลาใด
   - มีใครอยู่ที่บ้านบ้าง
   - และสงสัยว่าคนร้ายจะเป็นใคร

2. พนักงานสอบสวนต้องลง ปจว. ไปตรวจสถานที่เกิดเหตุ พร้อมกับมอบหน้าที่ให้นายตำรวจอื่นปฏิบัติหน้าที่แทน เมื่อไปถึงที่เกิดเหตุ
   1. ต้องตรวจสอบร่องรอยสถานที่เกิดเหตุเพื่อหาหลักฐาน เช่น
      - ที่เก็บทรัพย์
      - ทางเข้า ทางออกของคนร้าย
      - เครื่องมือของคนร้ายที่เก็บได้ ณ ที่เกิดเหตุมีอะไรบ้าง
      - คนร้ายเจาะ งัด เปิดประตูหรือหน้าต่างโดยวิธีการอย่างไร

   2. พิมพ์ลายนิ้วมือของคนร้ายที่ควรจะเก็บไว้ ต้องตรวจดูบริเวณ
      - หน้าต่าง
      - ประตูบ้าน
      - กระจก
      - ขวด
      - แก้วน้ำ
      - ตู้เย็น
      - สวิตซ์ไฟฟ้า
      - หลอดไฟฟ้า
      - ตู้เซฟ
      - เหล็กงัด
      - ฯลฯ

      ถ้าพบลายนิ้วมือ ต้องเก็บลายนิ้วมือไปตรวจสอบ

   3. การตรวจสถานที่เกิดเหตุ ต้องสังเกตการกระทำของคนร้ายบางประเภท เช่น เมื่อได้ทรัพย์สิ่งของแล้ว ชอบเปิดตู้เย็น ดื่มน้ำ ดื่มเบียร์ กินอาหารในตู้เย็น เช่น ผลไม้ เสร็จแล้วชอบถ่ายอุจจาระ ปัสสาวะไว้ในมุมบ้าน ชอบปีนกำแพงรั้วออกไปโดยไม่ออกทางประตู เป็นต้น

   4. บันทึกการตรวจสถานที่เกิดเหตุ ต้องระบุให้ชัดถึง
      - ลักษณะบ้าน เช่น บ้านชั้นเดียว สองชั้น เรือนไม้ ตึกคอนกรีต หลังเดียว หรือห้องแถว
      - บริเวณใกล้เคียง มีบ้านเรือนของใครอยู่บ้าง ห่างไกลกันเท่าไร
      - มีรั้วรอบขอบชิดหรือไม่ เป็นรั้วไม้ สังกะสี คอนกรีต หรืออย่างไร
      - สูงจากพื้นดินเท่าไร

3. บุคคลที่อยู่ในบ้าน โดยเฉพาะคนแปลกหน้าที่ไปขอพักอาศัยกับคนใช้ ต้องสอบสวนทุกคน โดยเฉพาะถ้าตรวจพบลายนิ้วมืออยู่ในบริเวณที่เกิดเหตุ บุคคลที่ถูกสงสัย ต้องพยายามจัดให้มีการพิมพ์ลายนิ้วมือเพื่อส่งไปตรวจสอบ

4. ถ้าจับผู้ต้องหาได้ ถ้าผู้ต้องหาปฏิเสธ ให้บันทึกไว้ ถ้าผู้ต้องหารับสารภาพ ให้สอบสวนอย่างละเอียด ให้ได้ความว่า
   - ในการไปลักทรัพย์ครั้งนี้ไปกับใคร
   - วัน เดือน ปี เวลา ที่กระทำ
   - เข้าไปในบ้านเขาอย่างไร เข้าทางไหน
   - เอาทรัพย์ไปได้อย่างไร
   - ออกจากบ้านเขาไปทางไหน
   - ไปโดยยานพาหนะอะไร
   - ทรัพย์สิ่งของที่ได้ไปอยู่ที่ไหน
   - สมัครใจพาพนักงานสอบสวนไปชี้ที่เกิดเหตุหรือไม่
   - พาพนักงานสอบสวนไปชี้ที่เกิดเหตุ ตามขั้นตอนก่อนเกิดเหตุ ขณะเกิดเหตุ หลังเกิดเหตุ ทำอย่างไร
