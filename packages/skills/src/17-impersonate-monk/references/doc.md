# การสอบสวนความผิดเกี่ยวกับแต่งกายเป็นภิกษุ (ในพระพุทธศาสนาโดยมิชอบ เพื่อให้บุคคลอื่นเชื่อว่าตนเป็นภิษุที่แท้จริง)

## 1. สอบสวนผู้กล่าวหาและพยานให้ได้ความว่า

1. ได้ทราบว่าพระภิษุ (ผู้ต้องหา) กระทำความผิดด้วยตนเอง หรือมีใครมาบอกให้
2. ถ้ามีผู้มาบอกกล่าว เล่าให้ฟัง ผู้นั้นเป็นใคร อยู่ที่ไหน หากไม่ควรเปิดเผยชื่อและที่อยู่ก็ไม่ต้องถามข้อนี้ เป็นเพียงบันทึกลงไปว่า มีผู้ที่ไม่ประสงค์ที่จะออกนามมาแจ้งให้ทราบว่า
3. ที่พบผู้ต้องหาอยู่ที่ไหน แต่งกายอย่างไร และกำลังทำอะไรอยู่
4. ผู้ต้องหาแต่งกายเป็นพระภิษุสงฆ์ หรือแต่งกายในศาสนาใด และศาสนานั้นๆ ทางราชการรับรองหรือไม่
5. เชื่อหรือไม่ว่า ผู้ต้องหาเป็นพระภิกษุ สามเณร หรือนักบวช นักพรตในศาสนานั้นจริงๆ ถ้าเชื่อได้ ทำความเคารพโดยยกมือไหว้ จับมือ ถวายหรือให้สิ่งของเป็น "ทาน" บ้างหรือไม่
6. ได้สอบถามผู้ต้องหาหรือไม่ว่า บวชหรืออุปสมบท บรรพชาที่วัดไหน วันเดือนปีใด ใครเป็นพระ อุปฌาย์ ใครเป็นพระคู่สวด (2 องค์) และมีพระอันดับกี่องค์ มีหนังสือรับรอง (ใบสุทธิ) ของพระอุปฌาย์ออกให้หรือไม่)
7. ผู้ต้องหาได้กระทำความผิดอะไรบ้าง มีพยานหลักฐานอย่างไร และมีอะไรบ้าง เป็นสิ่งที่ได้มาหรือมีไว้ครอบครองชอบด้วยกฎหมายหรือไม่
8. ขณะจับกุมผู้ต้องหาให้การอย่างไร (ปฏิเสธ ภาคเสธ รับสารภาพ)

## 2. สอบสวนผู้ต้องหา ถ้าผู้ต้องหาให้การรับสารภาพต้องให้ได้ความว่า

1. อายุ อาชีพ ที่อยู่ ที่เกิด
2. บวชหรืออุปสมบทที่วัดใด หมู่บ้าน ตำบล อำเภอ จังหวัดใด และเมื่อใด
3. ใครเป็นคนโกนศีรษะให้ เครื่องแต่งกายเช่น สบง จีวร อังสะ รัดปะคต บาตร หรือเครื่องอัฐบริบาลได้มาจากไหน ใครให้หรือซื้อมาเอง ซื้อมาจากร้านไหน ตลาดใด ตำบล อำเภอ จังหวัดใด ราคาเท่าไร
4. ใครเป็นพระอุปฌาย์ ใครเป็นพระคู่สวดและใครบ้างที่เป็นพระอันดับขณะที่บวช
5. มีใบสุทธิรับรองการบวชของพระอุปฌาย์หรือไม่
6. บวชแล้วจำพรรษาอยู่ที่วัดใด และได้เดินทางไปที่ไหนบ้าง การเดินทางออกจากวัดได้รับอนุญาตหรือบอกกล่าวเจ้าอาวาสวัดหรือไม่
