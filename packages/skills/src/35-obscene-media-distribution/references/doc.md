# การสอบสวนคดีมีไว้เพื่อประสงค์แห่งการค้า เพื่อจำหน่าย ทำให้แพร่หลาย ซึ่งแถบบันทึกภาพและหนังสือรูปภาพลามกอนาจาร

1. ต้องสอบสวนผู้กล่าวหาให้ได้ความว่า
   1. รับราชการ ยศใด ตำแหน่งใด มีหน้าที่อย่างไร
   2. ผลการสืบสวนล่วงหน้ามาก่อนเกี่ยวกับเรื่องนี้ได้ความอย่างไร
   3. เมื่อวัน เดือน ปี เวลาใด ได้พร้อมกับใครไปทำอะไร ที่ไหน และทำอย่างไร
   4. ผลการตรวจค้น และจับกุมได้ของกลางอะไรบ้าง ของกลางแต่ละอย่างแต่ละชนิดอยู่ที่ไหน อยู่ในลักษณะปิดบังซ่อนเร้นไว้หรือวางไว้อย่างเปิดเผย มีจำนวนแต่ละอย่างหรือแต่ละชนิดจำนวนเท่าไร อยู่ในความครอบครองของใคร หรือของกลางนั้นเป็นของใคร
   5. ผู้ต้องหามีกี่คน ชื่อ นามสกุลอะไร อยู่ที่ไหน สถานที่พบของกลางเป็นสถานที่ร้านค้า บ้านเรือนของตนเอง หรือเช่าจากคนอื่น หรือขออาศัยชั่วขณะ ของกลางเป็นของใคร มีไว้ทำไม ชั้นจับกุมผู้ต้องหารับสารภาพ ภาคเสธ หรือปฏิเสธ
   6. นำตัวผู้ต้องหา ของกลาง ส่งพนักงานสอบสวนที่ใด ทำการสอบสวน
   7. ได้ตรวจสอบของกลางต่อหน้าพนักงานสอบสวน ผู้ต้องหา พยาน ได้ความอย่างไร ได้บันทึกไว้หรือไม่
   8. เหตุเกิดที่ตำบล (แขวง) อำเภอ (เขต) จังหวัดใด
   9. เคยรู้จักหรือเคยมีสาเหตุโกรธเคืองกับผู้ต้องหาที่ถูกจับกุมมาก่อนหรือไม่
2. ต้องสอบสวนผู้ต้องหา (ถ้าให้การรับสารภาพ) ให้ได้ความว่า
   1. ที่รับสารภาพนั้น รับสารภาพด้วยความสมัครใจหรือไม่ มีผู้ใดบังคับ ขู่เข็ญ ให้คำมั่นสัญญา หลอกลวง หรือจูงใจหรือไม่
   2. ถูกเจ้าพนักงานตรวจค้น จับกุม ที่ไหน วัน เดือน ปี ใด
   3. เจ้าพนักงานยึดของกลางที่ใช้ในการกระทำผิด หรือที่มีไว้เป็นความผิดมีอะไรบ้าง กี่อย่าง กี่ชนิด จำนวนเท่าไร ของกลางนั้นเป็นของใคร ได้มาจากไหน มีไว้เพื่อประโยชน์อะไร
   4. สถานที่ที่เกิดเหตุเป็นของตนเอง หรือเช่าจากผู้อื่น หรือเป็นที่สาธารณะ
   5. กระทำกิจการคนเดียว หรือมีผู้อื่นร่วมด้วย ถ้ามีเป็นใคร อยู่ที่ไหน อายุเท่าไร ตำหนิรูปพรรณอย่างไร
   6. ได้ตรวจของกลางต่อหน้าผู้กล่าวหา พนักงานสอบสวน พยาน ได้ความว่าอย่างไร ตามที่พนักงานสอบสวนบันทึกไว้ใช่หรือไม่
   7. เคยต้องโทษหาหรือต้องโทษในคดีอาญามาก่อนหรือไม่
3. ต้องสอบสวนพยานให้ได้ความสอดคล้องต้องกันกับคำให้การของผู้กล่าวหา
