# การสอบสวนคดีความผิดเกี่ยวกับอาวุธปืน กระสุนปืน ฯ

ควรดำเนินการดังนี้

1. เป็นอาวุธปืน เครื่องกระสุนปืนชนิดใด จำนวนเท่าไร
2. อาวุธปืนมีทะเบียนจากนายทะเบียนอาวุธปืน (นายทะเบียนท้องที่) หรือไม่
   - หมายเลขทะเบียนและหมายเลขอาวุธปืนตรงกับทะเบียนหรือไม่
   - เป็นอาวุธปืนของใคร
   - จับได้หรือยึดได้จากใคร ที่ไหน เมื่อใด (วัน เวลา สถานที่เกิดเหตุ)
3. ผู้ที่มีไว้ในความครอบครอง
   - มีใบอนุญาตให้พกพาอาวุธปืนติดตัวไปในเมือง หมู่บ้าน ทางสาธารณะหรือไม่
   - หรือการนำพาอาวุธปืนติดตัวไปมีเหตุผลสมควร และมีความจำเป็นเร่งด่วนหรือไม่ อย่างไร
4. การนำพาไปหรือมีไว้ในความครอบครอง หรือพาไปในที่ชุมชน
   - ที่จัดให้มีขึ้นเพื่อการรื่นเริง มหรสพ หรือกิจกรรมอื่นใด
   - เป็นการกระทำโดยเปิดเผยหรือไม่
5. กระสุนปืนที่มีไว้ ใช้กับอาวุธปืนที่ได้รับอนุญาตหรือไม่
6. อาวุธปืน กระสุนปืน ที่มีไว้ใช้ เป็นอาวุธสำหรับการสงครามหรือไม่
7. ต้องส่งอาวุธปืน กระสุนปืน วัตถุระเบิด ที่จับหรือยึดได้ไปให้ผู้รู้หรือผู้เชี่ยวชาญตรวจพิสูจน์ว่า
   1. เป็นอาวุธปืน กระสุนปืน ที่ใช้ยิงได้หรือไม่
   2. เป็นอาวุธปืน กระสุนปืน ชนิดใด ขนาดใด ถ้ามีไว้ในความครอบครองโดยไม่ได้รับอนุญาต จะผิดกฎหมายหรือไม่ (พ.ร.บ.อาวุธปืนฯ)
   3. เป็นอาวุธปืน กระสุนปืน ที่นายทะเบียนไม่อาจออกใบอนุญาตให้ได้หรือไม่ หากมีไว้ในความครอบครอง จะผิดกฎหมายหรือไม่
8. การส่งอาวุธปืน กระสุนปืน ไปตรวจพิสูจน์ ควรส่งไป
   1. กองพิสูจน์หลักฐาน กรมตำรวจ (พฐ. ตร.)
   2. ในส่วนภูมิภาค
      - ส่งไปตรวจที่ กก.วท.บก.ภ. ……………
      - หรือ วท.กก.ภ.จว. ……………
      - หรือ เจ้าหน้าที่ พธ.กก.ภ.จว. ……………
      - หรือ ผบก.พธ.ตร.
   3. ถ้าอาวุธปืน กระสุนปืน หรือวัตถุระเบิดไม่มีปัญหา ให้ส่งหน่วยสรรพาวุธของทหารที่ใกล้ และเจ้าหน้าที่ฝ่ายทหารที่มีความรู้ความชำนาญตรวจพิสูจน์ (ถ้าไม่จำเป็น ไม่ต้องส่งตรวจตามข้อนี้)
   4. การส่งอาวุธปืน กระสุนปืน วัตถุระเบิดไปตรวจพิสูจน์ ต้องจัดการส่งตามแบบฟอร์ม การบรรจุหีบห่อตามระเบียบของกรมตำรวจ (ป.เกี่ยวกับคดี ลักษณะ 15 บทที่ 5)
