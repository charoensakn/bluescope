# การสอบสวนคดีความผิดตาม พ.ร.บ.ควบคุมการฆ่าสัตว์ และจำหน่ายสัตว์

การสอบสวนควรดำเนินการดังต่อไปนี้

1. ต้องสอบสวนผู้กล่าวหาให้ได้ความว่า ก่อนมีการจับกุมคดีนี้ได้มีการสืบสวนมาก่อนหรือไม่ ได้ความประการใด หรือมีสายลับผู้ไม่ประสงค์จะออกนามนำความมาบอก บอกว่าอย่างไร ให้ใครไปสืบสวน เฝ้าจุดตามที่รับแจ้ง ขณะไปจับกุมผู้ต้องหา ไปกับใครบ้าง พบผู้ต้องหากี่คน ชื่ออะไร ผู้ต้องหากำลังทำอะไร อยู่ที่ไหน (เช่น ผู้ต้องหาฆ่ากระบืออยู่ที่โรงฆ่าสัตว์หรือใต้ร่มไม้) ผู้ต้องหามีใบอนุญาตให้ฆ่าจากพนักงานเจ้าหน้าที่หรือไม่ ยึดของกลางได้อะไรบ้าง ขณะจับกุมผู้ต้องหาให้การรับสารภาพหรือปฏิเสธ หรือภาคเสธว่าอย่างไร เคยมีสาเหตุโกรธเคืองกับผู้ต้องหามาก่อนหรือไม่ เหตุเกิดที่ไหน เมื่อไหร่
2. ต้องสอบสวนพยานที่รู้เห็นเหตุการณ์และพยานผู้จับกุมฆ่า รู้เห็นอย่างไร ผู้จับกุมได้จับกุมผู้ต้องหาพร้อมของกลางอะไรบ้าง ตำหนิรูปพรรณอย่างไร ชั้นจับกุมผู้ต้องหาให้การอย่างไร
3. การสอบสวนผู้ต้องหา ถ้าผู้ต้องหาให้การรับสารภาพต้องสอบสวนให้ได้รายละเอียดของการกระทำผิด เช่น สัตว์ที่ฆ่าเป็นสัตว์อะไร สัตว์นั้นเป็นของใคร ถ้าเป็นสัตว์พาหนะ เช่น โค กระบือ มีหลักฐานการได้มาอย่างไร มีรูปพรรณหรือไม่ ถ้าซื้อมาจากคนอื่นได้จดทะเบียนการโอนกรรมสิทธิ์ต่อนายทะเบียนหรือไม่ การฆ่าสัตว์เพื่ออะไร มีใบอนุญาตให้ฆ่าจากพนักงานเจ้าหน้าที่หรือไม่ ฆ่าที่ไหน เมื่อไหร่ ใครช่วยเหลือในการฆ่าและการชำแหละเนื้อ ของกลางที่เป็นเครื่องมือในการฆ่ามีอะไรบ้าง เป็นของตนเองหรือหยิบยืมมาจากคนอื่น
