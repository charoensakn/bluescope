# การสอบสวนคดียักยอกทรัพย์

ควรดำเนินการดังนี้

1. เมื่อมีผู้แจ้งความร้องทุกข์ ร้อยเวรหรือพนักงานสอบสวนจะต้องรีบสอบถามข้อเท็จจริงที่เกิดขึ้นอย่างละเอียด รวมถึงวันเวลา สถานที่เกิดเหตุ แล้วพิจารณาว่าเป็นคดีแพ่งหรือคดีอาญา ถ้าเป็นคดีแพ่ง ให้แนะนำผู้แจ้งไปฟ้องร้องต่อศาลเอง เพราะอยู่นอกเหนืออำนาจของพนักงานสอบสวนจะจัดการให้ได้ แล้วลง ปจว. ไว้เป็นหลักฐาน ถ้าเป็นคดีอาญาความผิดฐานยักยอกทรัพย์ ให้รับคำร้องทุกข์หรือกล่าวโทษไว้ คดียักยอกทรัพย์เป็นคดีความผิดอันยอมความได้ จะต้องบันทึกการมอบคดีให้พนักงานสอบสวนดำเนินคดีจนถึงที่สุด
2. สอบสวนผู้เสียหายหรือผู้กล่าวหาให้ได้ความว่า
   - วันเวลา สถานที่เกิดเหตุที่มอบทรัพย์และที่รู้ว่ามีการกระทำผิดเกิดขึ้น
   - ทรัพย์ที่ถูกประทุษร้ายนั้นเป็นอย่างไร ลักษณะ ตำหนิรูปพรรณของทรัพย์ รวมทั้งราคาทรัพย์
   - ทรัพย์นั้นเป็นของตนเอง หรือผู้อื่นมีส่วนเป็นเจ้าของอยู่ด้วย
   - ได้มอบทรัพย์นั้นให้กับใคร อยู่ที่ไหน รู้จักกันหรือสนิทสนมกันอย่างไร นานเท่าไร มีตำหนิรูปพรรณอย่างไร
   - เป็นผู้ดูแลรักษา หรือเก็บ หรือจัดการทรัพย์นั้น
   - มีพยานหลักฐานรู้เห็นเหตุการณ์อย่างไร
   - ถ้าจับผู้ต้องหาและของกลางได้ ต้องให้ผู้เสียหายและพยานดูตัวผู้ต้องหา หรือชี้ตัวผู้ต้องหาแล้วแต่กรณี กับให้ผู้เสียหายดูทรัพย์ที่ยึดได้ว่าเป็นของผู้เสียหายใช่หรือไม่
   - ถ้าผู้ต้องหาไม่เถียงกรรมสิทธิ์ ควรมอบทรัพย์คืนให้เจ้าของ นำไปเก็บรักษาไว้ ให้ลงชื่อรับไปใน ปจว. สมุดยึดทรัพย์ บัญชีของกลางคดีอาญา
3. ต้องกรอกรายละเอียดตำหนิรูปพรรณทรัพย์หายให้ครบถ้วน แล้วจัดการส่งไปยังหน่วยงานที่เกี่ยวข้องประกาศสืบจับต่อไป
   - ส่วนกลาง (ตำรวจนครบาล) ส่งไปยังกองทะเบียนประวัติอาชญากร
   - ส่วนภูมิภาค (ตำรวจภูธร) ส่งไปยัง สว.กองวิทยาการจังหวัด หรือ กก.วท.บก.ภ. (หมายเลข) ก็ได้
4. เมื่อทรัพย์ที่ถูกประทุษร้ายได้คืนมาทั้งหมดหรือแต่บางส่วน ต้องลงบัญชีทรัพย์ที่ถูกประทุษร้ายได้คืนแล้ว แจ้งให้ สว.กองวิทยาการจังหวัด หรือ กก.วท.บก.ภ. (หมายเลข) หรือทะเบียนประวัติอาชญากรทราบ
5. ถ้าผู้เสียหายรับสารภาพ ให้สอบสวนให้ได้ความว่า
   - วัน เดือน ปี ใด ได้รับมอบหมายทรัพย์จากผู้เสียหาย
   - ทรัพย์นั้นมีอะไรบ้าง
   - ได้รับมอบหมายให้เป็นผู้ดูแลรักษา หรือเก็บ หรือจัดการทรัพย์นั้นอย่างไรบ้าง
   - ทรัพย์นั้นเอาไว้ใช้เองหรือเอาไว้เป็นอาณาประโยชน์ของตนเองหรือของใคร อยู่ที่ไหน เมื่อไร
   - รู้จักกับผู้เสียหายอย่างไร
   - ขณะที่ผู้เสียหายมอบหมายทรัพย์นั้น มีใครรู้เห็นบ้าง
   - เคยต้องหาต้องโทษคดีอาญาถึงจำคุกมาก่อนหรือไม่
