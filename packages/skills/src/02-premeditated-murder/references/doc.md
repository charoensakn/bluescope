# การสอบสวนคดีฆ่าผู้อื่นโดยเจตนาและไตร่ตรองไว้ก่อน

1. ฆ่าผู้อื่นโดยเจตนาและไตร่ตรองไว้ก่อนเป็นคดีอุกฉกรรจ์ ต้องรายงานด่วนตามระเบียบ และพนักงานสอบสวนต้องทำคดีด้วยตนเอง
2. รีบออกไปสืบสวน–สอบสวนยังที่เกิดเหตุ โดยดำเนินการชั้นต้นดังนี้
   1. ตรวจหาร่องรอยสถานที่เกิดเหตุ
   2. ค้นหาและเก็บพยานหลักฐาน เช่น ลายนิ้วมือ รอยเท้า เลือด เส้นผม ขน เสื้อผ้า รองเท้า เครื่องมือ อาวุธปืน มีด หอก ดาบ แหลน หลาว
   3. ทำแผนที่สังเขปประกอบคดี
   4. ถ่ายรูปสถานที่เกิดเหตุและพยานหลักฐานสำคัญ
   5. บันทึกการตรวจสถานที่เกิดเหตุอย่างละเอียด
   6. บันทึกวัน เวลา สถานที่เกิดเหตุให้แน่นอน
3. ร่วมกับแพทย์ชันสูตรพลิกศพ โดยพนักงานสอบสวนเป็นผู้แจ้งแพทย์
4. ตรวจสภาพศพอย่างละเอียด เช่น ท่าศพ การแต่งกาย รอยฟัน แทง ยิง ทางเข้าทางออกกระสุน ความกว้าง–ยาวของบาดแผล ตำแหน่งบาดแผล สาเหตุการตาย พร้อมถ่ายรูปและพิมพ์ลายนิ้วมือผู้ตาย
5. มอบศพให้ญาติหลังชันสูตร และบันทึกการรับศพไว้เป็นหลักฐาน
6. จดชื่อผู้พบศพคนแรกเพื่อสอบสวนเป็นพยาน
7. สอบสวนพยานบริเวณที่เกิดเหตุและผู้ใกล้ชิด รวมถึงผู้กล่าวหา เช่น บิดา มารดา หรือญาติของผู้ตาย
8. สืบสวนความประพฤติของผู้ตาย โดยสอบกำนัน ผู้ใหญ่บ้าน หรือบุคคลที่เชื่อถือได้ในหมู่บ้าน
9. จัดสายตรวจออกสืบสวนเพื่อทราบตัวคนร้าย
10. หากจับผู้ต้องหาได้และรับสารภาพ ต้องสอบละเอียดเกี่ยวกับ
    - สาเหตุการฆ่า
    - อาวุธที่ใช้
    - ผู้ร่วมกระทำ
    - การกระทำก่อน–ขณะ–หลังเกิดเหตุ
    - การหลบหนี
    - คำให้การชั้นจับกุม
11. หากของกลางเป็นปืน ต้องตรวจ
    - ชนิดปืน
    - จำนวนกระสุน
    - ปลอกกระสุนที่พบ
    - การยิงมากี่นัด
    - ส่งพิสูจน์ที่ พฐ.ตร. ว่าเป็นปืนที่ใช้ยิงจริงหรือไม่
    - ตรวจหัวกระสุนจากศพหรือที่เกิดเหตุว่าใช้ยิงจากปืนกระบอกเดียวกันหรือไม่
    - ตรวจทะเบียนปืนว่าเป็นของใคร
12. หากของกลางเป็นมีด หอก ดาบ แหลน หลาว ไม้ ต้องตรวจ
    - ความกว้าง–ยาว
    - ส่วนที่เป็นไม้–เหล็ก
    - เส้นผ่าศูนย์กลาง
    - น้ำหนัก
    - คราบโลหิต
    - ตรวจเสื้อผ้า–รองเท้าผู้ต้องหาว่ามีคราบเลือดหรือไม่
    - ส่งตรวจพิสูจน์ว่าเลือดตรงกับผู้ตายหรือไม่
13. นำผู้ต้องหาชี้สถานที่เกิดเหตุอย่างละเอียด ทั้งก่อนเกิดเหตุ ขณะเกิดเหตุ และหลังเกิดเหตุ
14. หากผู้ต้องหาขอขมาโทษผู้เสียหายหรือญาติ ให้พนักงานสอบสวนอำนวยความสะดวก พร้อมบันทึกและถ่ายรูปไว้
15. รูปถ่ายและบันทึกสถานที่เกิดเหตุ ต้องให้ผู้ต้องหาและพยานตรวจดู หากถูกต้องให้ลงชื่อรับรอง
16. พยานทุกคนที่สอบสวนไว้
    - หากรู้จักผู้ต้องหามาก่อน ให้จัดดูตัว
    - หากไม่รู้จัก ให้จัดชี้ตัวผู้ต้องหาทุกคน พร้อมบันทึกและถ่ายรูป
    - การชี้ตัวต้องทำก่อนนำผู้ต้องหาไปชี้ที่เกิดเหตุ
    - จากนั้นสอบสวนพยานเพิ่มเติม
