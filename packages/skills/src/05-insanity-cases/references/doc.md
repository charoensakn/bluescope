# การสอบสวนคดีกระทำผิดในเวลาวิกลจริต

## 1. การสอบสวนผู้ต้องหา

หากผู้ต้องหาสามารถให้การได้ ให้สอบถามประเด็นต่อไปนี้

1. ข้อมูลประจำตัว
   - ชื่อ–นามสกุล
   - สัญชาติ
   - บิดา–มารดา
   - อายุ
   - อาชีพ
   - ที่อยู่
   - สถานที่เกิด
   - จำสิ่งเหล่านี้ได้เหมือนบุคคลปกติหรือไม่
   - มีบัตรประชาชนหรือหลักฐานอื่นยืนยันตัวตนหรือไม่

2. ความสามารถในการตอบคำถาม
   - ตอบคำถามคล่องแคล่วเหมือนคนปกติหรือไม่
   - สังเกตท่าทาง นัยน์ตา ริมฝีปาก มือสั่น
   - มีอาการเหมือนคนมึนเมา สุรา หรือยาเสพติดหรือไม่

3. การกระทำที่เกิดขึ้น
   - ทำอะไร กับใคร ที่ไหน
   - มีจุดประสงค์อะไร
   - เคยรู้จักผู้เสียหายหรือผู้กล่าวหามาก่อนหรือไม่

4. วิธีการกระทำ
   - ใช้อะไรทำ
   - ระหว่างทำรู้สึกตัวหรือไม่ว่ากำลังทำอะไร

5. การใช้สารเสพติด
   - ดื่มสุรา สูบบุหรี่ กัญชา หรือยาเสพติดใดบ้าง

6. ประวัติการรักษา
   - เคยป่วยเป็นโรคอะไร
   - เคยเข้ารักษาที่ไหน กี่วัน เมื่อใด

## 2. การสอบสวนพยาน ญาติพี่น้อง หรือผู้ใกล้ชิด

1. รู้จักผู้ต้องหามาก่อนหรือไม่ มีความเกี่ยวข้องอย่างไร
2. ผู้ต้องหาเคยเป็นคนวิกลจริตหรือไม่ เคยรักษาที่ไหน นานเท่าใด
3. ผู้ต้องหาอยู่ในความอุปการะของใคร (อดีต–ปัจจุบัน)
4. มีหลักฐานเกี่ยวกับการรักษาตัวหรือไม่

## 3. พยานเอกสาร

- ตรวจสอบประวัติการรักษาตัวของผู้ต้องหาจากโรงพยาบาลหรือสถานพยาบาลที่เคยเข้ารับการรักษา

## 4. การส่งตรวจรักษา

- ส่งผู้ต้องหาไปตรวจหรือรักษาที่โรงพยาบาลโรคจิต จิตเวช หรือแพทย์เฉพาะทางโรคประสาท
- ระหว่างส่งตรวจ ให้ **งดการสอบสวนผู้ต้องหา**
- ระยะเวลาที่อยู่โรงพยาบาล **ไม่นับเป็นการควบคุมตัว**
- การสอบสวนพยานหลักฐานอื่นยังคงดำเนินต่อไปได้

## 5. สอบสวนแพทย์

สอบถามแพทย์เกี่ยวกับผลการตรวจรักษา เช่น

- ผู้ต้องหาเป็นโรคจิตประสาทหรือไม่
- มีจิตบกพร่อง โรคจิต หรือจิตฟั่นเฟือนหรือไม่
- ระหว่างเกิดเหตุสามารถรู้ผิดชอบหรือยับยั้งตนเองได้หรือไม่

## 6. ประเด็นที่ควรพิจารณาในการสอบสวน

1. ผู้ต้องหามีจิตใจวิปลาสหรือวิกลจริตอันเกิดจากโรคหรือไม่
2. อาการถึงขั้นไม่สามารถรู้รับผิดชอบหรือยับยั้งตนเองได้หรือไม่

## 7. ผลการสอบสวนตามพยานหลักฐาน

หากได้ข้อเท็จจริงว่า

- ผู้ต้องหากระทำความผิดในขณะไม่สามารถรู้รับผิดชอบตนเองได้
- ตามประมวลกฎหมายอาญา มาตรา 65

ให้ **สรุปสำนวนและสั่งไม่ฟ้องผู้ต้องหา** ตามข้อกล่าวหา
