# การสอบสวนคดีความผิดฐานรับของโจร

1. เมื่อได้รับคำร้องทุกข์จากผู้เสียหาย หรือได้ทราบว่ามีคดีความผิดฐานรับของโจร เจ้าพนักงานตำรวจจะต้องรีบไปสืบสวนยังที่เกิดเหตุเพื่อให้ได้พยานหลักฐาน บางคดีพนักงานสอบสวนจะเดินทางไปสืบสวนด้วยตนเอง โดยร่วมกับเจ้าหน้าที่ตำรวจซึ่งส่งไปสืบสวนล่วงหน้าแล้ว ก่อนอื่นพนักงานสอบสวนต้องนึกทบทวนถึงองค์ประกอบการกระทำความผิดฐานรับของโจร คือ
   1. ผู้ใดได้กระทำดังต่อไปนี้
      1. ช่วยซ่อนเร้น
      2. ช่วยจำหน่าย
      3. ช่วยเอาไปเสีย
      4. ซื้อ
      5. รับจำนำ หรือ
      6. รับด้วยประการใด
   2. ซึ่งทรัพย์อันได้มาโดยการกระทำความผิด เฉพาะในฐานความผิดดังต่อไปนี้
      1. ลักทรัพย์ (ม.334,335)
      2. วิ่งราวทรัพย์ (ม.336)
      3. กรรโชก (ม.337)
      4. รีดเอาทรัพย์ (ม.338)
      5. ชิงทรัพย์ (ม.339)
      6. ปล้นทรัพย์ (ม.340)
      7. ฉ้อโกง (ม.341-348)
      8. ยักยอก (ม.352-356)
      9. เจ้าพนักงานยักยอก (ม.147)

   ผู้นั้นกระทำความผิดฐานรับของโจร

2. การสอบสวนต้องสอบผู้เสียหายถึงรายละเอียดของทรัพย์ที่สูญหายหรือถูกประทุษร้าย ทรัพย์นั้นเป็นอะไร ชนิดใด ราคาเท่าไร ลักษณะตำหนิรูปพรรณของทรัพย์อย่างละเอียด รวมทั้งหลักฐานต่าง ๆ อันจะนำมาประกอบยืนยันได้ว่าตนเป็นเจ้าของ หรือผู้อื่นเป็นเจ้าของรวมอยู่ด้วย
3. ควรสอบสวนถึงมูลฐานความผิดว่าอยู่ในฐานใดในฐานความผิด (ตาม ป.อาญา มาตรา 157) หรือไม่
4. วัน เดือน ปี เวลา สถานที่เกิดเหตุ ต้องสอบสวนให้ชัดเจน
5. ในกรณีที่จับตัวผู้ต้องหาได้พร้อมทั้งของกลาง
   1. กรณีผู้ต้องหาปฏิเสธ ให้สรุปสำนวนการสอบสวน แล้วมีความเห็นควรสั่งฟ้องหรือควรสั่งไม่ฟ้องตามพยานหลักฐานที่ปรากฏ
   2. กรณีผู้ต้องหาให้การรับสารภาพ จะต้องสอบสวนให้ได้รายละเอียดว่า ทรัพย์ของกลางที่เจ้าพนักงานยึดได้จากผู้ต้องหานั้น ผู้ต้องหาได้มาจากใคร ที่ไหน เมื่อใด โดยวิธีใด มีใครรู้เห็นบ้าง และมีหลักฐานอย่างไร (การสอบสวนผู้ต้องหาจะต้องถามในหัวข้อ วัน เวลา ราคา สถานที่ และเจตนาในการได้มาซึ่งทรัพย์นั้น รวมทั้ง ทำลาย ดัดแปลง เปลี่ยนสภาพไปจากเดิม) ให้นำผู้ต้องหาไปชี้ที่เกิดเหตุในการรับของกลางนั้น พร้อมทั้งถ่ายรูป ทำแผนที่สังเขป บันทึกการตรวจสถานที่เกิดเหตุไว้
6. ควรสังเกตกิริยาท่าทางผู้ต้องหา และการให้ความสะดวก ความร่วมมือในการตรวจค้นหาของกลาง การนำของกลางมามอบให้ และการบอกความจริงในครั้งแรก
7. ควรถามผู้ต้องหาและพยานด้วยว่า “ผู้ต้องหาทราบหรือไม่ว่า ทรัพย์ของกลางนั้น เป็นทรัพย์ที่ได้มาจากการกระทำความผิด (ซึ่งเข้าฐานหนึ่งฐานใดในฐานความผิด)”
8. สำนวนการสอบสวน พนักงานสอบสวนมักจะตั้งข้อหา “รับของโจร” ควบคู่ไปกับข้อหา “ลักทรัพย์” ดังนั้น จึงเห็นอยู่เสมอว่า เมื่อตั้งข้อหา “ลักทรัพย์” แล้ว มักจะตั้งข้อหา “รับของโจร” ไปด้วย เช่น “ลักทรัพย์หรือรับของโจร” ทั้งนี้เป็นการตั้งข้อหาภายใต้อำนาจของ ป.วิ.อาญา มาตรา 192 วรรคสาม ตามนัยคำพิพากษาฎีกาที่ 1109/2527
9. หากข้อเท็จจริงตามพยานหลักฐานแน่ชัดว่าเป็นคดีลักทรัพย์หรือเป็นคดีรับของโจร ก็ควรจะพิจารณาดำเนินคดีในข้อหาใดข้อหาหนึ่ง ไม่จำเป็นที่จะต้องตั้งข้อหาว่า “ลักทรัพย์หรือรับของโจร” (ตามนัยคำพิพากษาฎีกาที่ 819/2513)
10. วัน เวลา สถานที่เกิดเหตุคดีลักทรัพย์กับคดีรับของโจรนั้น จะไม่ตรงกัน ดังนั้น การสอบสวนจะต้องเกี่ยวโยงหรือเกี่ยวพันกันหลายท้องที่ ทั้งนี้แล้วแต่ข้อเท็จจริงที่เกิดขึ้น ดังนั้นจึงต้องสอบสวนให้ชัดเจนตามข้อ 4.
