# การสอบสวนคดีให้ทรัพย์สินแก่เจ้าพนักงานเพื่อจูงใจให้ไม่กระทำการอันมิชอบด้วยหน้าที่

## 1. การสอบสวนผู้กล่าวหา ควรสอบสวนให้ได้ความว่า

1. รับราชการเป็นอะไร ยศใด ตำแหน่งใด มีอำนาจหน้าที่อย่างไรบ้าง
2. ได้ไปปฏิบัติหน้าที่อะไร ที่ไหน เมื่อไร ไปกับใคร ตามคำสั่งของผู้ใด
3. ขณะปฏิบัติหน้าที่ได้พบกับใคร ที่ไหน และมีเหตุการณ์อะไรเกิดขึ้น เช่น มีผู้ต้องหาให้สินบน ให้อะไร จำนวนเท่าไร เพื่อประสงค์อะไร ตัวผู้ต้องหาได้กระทำความผิดอะไร มีพยานหลักฐานอย่างไรบ้าง
4. เหตุเกิดวัน เดือน ปีใด และที่ไหน
5. ผู้กล่าวหาเคยรู้จักกับผู้ต้องหา และมีสาเหตุโกรธเคืองกันมาก่อนหรือไม่

## 2. การสอบสวนผู้ต้องหา ควรสอบสวนให้ได้ความว่า

1. สอบประวัติความเป็นอยู่ อาชีพ และการศึกษาโดยสังเขป
2. ได้กระทำความผิดอะไร ที่ไหน เมื่อไร
3. ให้ทรัพย์สินอะไรกับเจ้าพนักงาน มีความประสงค์เพื่ออะไร มีใครอยู่บริเวณที่เกิดเหตุบ้าง
4. เคยต้องหาต้องโทษในคดีอาญามาก่อนหรือไม่

## 3. การสอบสวนพยาน ควรสอบสวนให้ได้ความว่า

1. พยานไปทำอะไร ที่ไหน กับใคร จึงได้เห็นผู้ต้องหาให้ทรัพย์สินหรือสินบนแก่เจ้าพนักงาน
2. เจ้าพนักงานนั้นเป็นข้าราชการอะไร อยู่ที่ไหน ทราบหรือไม่ว่ามีอำนาจหน้าที่อย่างไร และพนักงานผู้นั้นทำอะไรอยู่กับใคร
3. พยานได้เห็นเหตุการณ์ในระยะเท่าใด ถ้าเป็นกลางคืน อาศัยแสงสว่างจากอะไร
4. เคยรู้จักกับเจ้าพนักงาน ผู้กล่าวหา หรือผู้ต้องหามาก่อนหรือไม่
