# การสอบสวนความผิด เอาตัวเด็กอายุยังไม่เกินสิบห้าปีไปหน่วงเหนี่ยวหรือกักขัง เพื่อให้ได้มาซึ่งค่าไถ่ พรากเด็กอายุยังไม่เกินสิบห้าปีไปเสียจากบิดามารดา

## 1. การสอบสวนผู้เสียหายหรือผู้กล่าวหา ควรสอบสวนให้ได้ความว่า

1. เป็นบิดามารดา หรือผู้ปกครองผู้ดูแลเด็กที่ถูกพาตัวไปใช่หรือไม่ หรือมีหลักฐานอย่างไร
2. ให้นำสำเนาทะเบียนบ้าน สำเนาสูติบัตรของเด็ก (ถ้ามี) ติดสำนวนด้วย เพราะเกี่ยวกับอายุของเด็ก
3. ขณะเกิดเหตุเด็กคนกลางอยู่ในความดูแลของผู้กล่าวหาหรือไม่ หรืออยู่ในความดูแลของใคร
4. ได้อนุญาตให้เด็กไปที่ใด เป็นอิสระหรือไม่ อย่างไร
5. ทราบจากใครว่าเด็กได้หายไปอย่างไร เมื่อใด ถ้าเด็กได้เล่าให้ฟัง เล่าว่าอย่างไรบ้าง ต้องสอบให้ละเอียด
6. ก่อนเกิดเหตุ คือก่อนเด็กหายไป มีใครรู้เห็นบ้างว่า มีใครไปติดต่ออะไรกับใคร และว่าอย่างไร มีใครบ้างที่เห็นคนพาเด็กไปที่ไหน เห็นตรงไหน เห็นอย่างไร จำตำหนิรูปพรรณของผู้ที่พาเด็กไปได้หรือไม่ เคยพบเห็นมาก่อนหรือไม่ ขณะที่เห็นผู้พาเด็กไป ไปพาหนะอะไร คนพาไปกี่คน
7. หลังจากที่เด็กได้ถูกพาตัวไป มีใครมาติดต่อว่าอย่างไรบ้าง ขู่เข็ญเรียกร้องทรัพย์สินอะไรเป็นเครื่องตอบแทน ให้ไปพบกันที่ไหน วัน เดือน ปี เวลา ที่นัดพบ
8. การติดต่อ ได้ติดต่อกับใคร โดยวิธีใด เช่น มีนายหน้ามาติดต่อ ติดต่อทางโทรศัพท์ ติดต่อทางจดหมาย และอื่น ๆ มีหลักฐานอย่างไรบ้าง ถ้าไม่ได้ตกลงตามข้อเรียกร้อง ขู่เข็ญว่าจะทำอย่างไรกับเด็กคนกลางบ้าง
9. ได้ไปแจ้งให้ใครทราบบ้าง และได้ดำเนินการตามคำเรียกร้องของผู้พาเด็กไปอย่างไรบ้าง
10. ได้เด็กคืนมาอย่างไร วัน เดือน ปี เวลา ที่ได้พบเด็กคนกลาง เด็กคนกลางมีสภาพเป็นอย่างไร มีร่องรอยถูกทำร้ายที่ไหนบ้าง ได้พาเด็กไปให้แพทย์ตรวจรักษาที่ใด สุขภาพจิตของเด็ก บาดแผลที่พบเห็นเป็นอย่างไร แพทย์รักษากี่วัน หาย
11. ถ้าได้เสียทรัพย์สินไปแล้ว ได้คืนหรือไม่ จำนวนเท่าไร
12. เคยรู้จักหรือมีสาเหตุโกรธเคืองกับผู้ที่พาเด็กไปมาก่อนหรือไม่

## 2. การสอบสวนเด็กคนกลางที่ถูกลักพาตัวไป ถ้าสามารถให้การได้ ควรสอบสวนให้ได้ความว่า

1. ขณะที่ถูกลักพาไปนั้น อยู่กับใคร ที่ไหน และกำลังทำอะไรอยู่
2. คนที่มาพาตัวไป เป็นหญิงหรือชาย มาด้วยพาหนะอะไร มากี่คน มีอาวุธอะไรบ้างหรือไม่
3. คนที่พาไปพูดจาข่มขู่ว่าอย่างไร จะพาไปไหน ถูกกักขังหน่วงเหนี่ยวไว้ที่ไหน ใครทำร้ายร่างกาย เช่น ทุบตี ใช้เชือกมัด อย่างไรบ้าง หรือไม่ ถูกกักขังหน่วงเหนี่ยวไว้ที่ไหน เปลี่ยนสถานที่บ้างหรือไม่ คนควบคุมดูแล ข่มขู่ ทุบตี ทำร้ายที่ไหนบ้าง เป็นใคร จำได้หรือไม่
4. ขณะที่ถูกกักขัง หน่วงเหนี่ยว และถูกทำร้าย ทารุณ มีใครเห็นบ้างหรือไม่
5. ใครเป็นผู้ช่วยเหลือ วัน เดือน ปีใด จึงได้พบกัน

## 3. การสอบสวนพยานก่อนเกิดเหตุ ควรให้ได้ความว่า

1. เคยรู้จักบิดามารดา ผู้ปกครอง หรือผู้ดูแลเด็กคนกลางมาก่อนหรือไม่ ถ้าเคยรู้จัก นานเท่าไร ที่ไหน สาเหตุที่รู้จักกัน
2. เคยรู้จักหรือเคยเห็นหน้าผู้ที่ลักพาเด็กคนกลางมาก่อนหรือไม่ ถ้าเคย เห็นที่ไหน เมื่อไร
3. เคยเล่าเรื่องที่ได้พบเห็นเหตุการณ์ให้ได้ฟังบ้าง
4. เคยมีสาเหตุกับผู้ลักพาเด็กคนกลางมาก่อนหรือไม่

## 4. การสอบสวนพยานที่เห็นเหตุการณ์ ขณะที่เด็กคนกลางถูกควบคุม กักขัง หน่วงเหนี่ยวไว้ ให้ได้ความว่า

1. รู้เห็นอย่างไรเกี่ยวกับกรณีที่มีผู้ลักพาเด็กไป รู้จักกับผู้ลักพาเด็กไปมาก่อนหรือไม่
2. วัน เวลา สถานที่ที่ได้พบเห็น ได้พบเห็นอย่างไรบ้าง ได้เล่าหรือบอกให้ใครทราบบ้างหรือไม่

## 5. การสอบสวนผู้ต้องหา ถ้าผู้ต้องหาให้การรับสารภาพ ควรสอบสวนให้ละเอียด และให้ได้ความว่า

1. ผู้ต้องหาเคยรู้จักกับผู้กล่าวหา หรือผู้เสียหายมาก่อนหรือไม่ ถ้าเคยรู้จัก ที่ไหน เมื่อใด
2. เคยเห็นเด็กซึ่งเป็นบุตรของผู้กล่าวหามาก่อนหรือไม่ ถ้าเคยเห็น เห็นที่ไหน ชื่ออะไร มีใครปกครองหรือดูแล
3. สาเหตุจงใจที่ต้องจับเด็กไปเรียกค่าไถ่
4. วางแผนกับใคร ที่ไหน กี่คน แต่ละคนทำหน้าที่อย่างไร
5. ลักพาเด็กไปแล้ว นำไปไว้ที่ไหน ให้ใครควบคุม กักขัง หน่วงเหนี่ยวไว้
6. เรียกร้องเอาทรัพย์สินเป็นค่าไถ่ เด็กคนกลางกับใคร ทรัพย์สินอะไร เป็นมูลค่าเท่าไร ขู่เข็ญว่าจะฆ่าใคร ถ้าไม่ให้ค่าไถ่ และถ้าแจ้งความตำรวจ จะทำอะไรกับใคร
7. เรียกร้องเอาทรัพย์สิน ขู่เข็ญ ทางจดหมาย โทรศัพท์ หรือให้ใครเป็นผู้ติดต่อ
8. ผู้ต้องหาถูกจับกุมตัวที่ไหน เมื่อไร ได้ของกลางอะไรบ้าง ชั้นจับกุมให้การกับเจ้าหน้าที่ตำรวจอย่างไร สมัครใจพาพนักงานสอบสวนไปชี้ที่เกิดเหตุได้หรือไม่

## 6. ให้สอบสวนเจ้าหน้าที่ตำรวจที่ทำการสืบสวนและติดตามจับกุมผู้ต้องหา พร้อมด้วยเด็กคนกลางคืน ให้ได้ความว่า

1. ทราบเรื่องคดีเกิดขึ้นจากใคร ที่ไหน และได้ปฏิบัติตามขั้นตอนอย่างไร กับใครบ้าง
2. จับกุมตัวผู้ต้องหาได้กี่คน ที่ไหน เมื่อไร ได้ของกลางอะไร ที่ไหน พบเด็กคนกลางที่ไหน เมื่อไร อยู่ในสภาพอย่างไร ได้ช่วยเหลือขั้นต้นอย่างไรบ้าง
3. ชั้นจับกุม ผู้ต้องหาให้การอย่างไร

## 7. พนักงานสอบสวนต้องไปตรวจสถานที่เกิดเหตุ

1. พนักงานสอบสวนต้องไปตรวจสถานที่เกิดเหตุ 3 แห่ง คือ
   1. บ้านของผู้กล่าวหา หรือที่เด็กคนกลางถูกลักพาตัวไป
   2. สถานที่ที่จับตัวผู้ต้องหาได้ พร้อมของกลาง
   3. บ้านที่พบเด็กคนกลาง สภาพของบ้านที่ถูกหน่วงเหนี่ยว กักขัง
2. ต้องทำแผนที่สังเขปที่เกิดเหตุทั้ง 3 แห่ง พร้อมบันทึกการตรวจสถานที่เกิดเหตุประกอบ
3. สถานที่เกิดเหตุแต่ละแห่ง อยู่ห่างกันเท่าไร ตรอก ซอย ถนน หมู่บ้าน ตำบล/แขวง อำเภอ/เขต จังหวัดใด (บางคดีเกิดหลายท้องที่เกี่ยวพันกัน)
4. พนักงานสอบสวนต้องหาพยานหลักฐานและเก็บของกลาง เช่น
   - สำเนาทะเบียนบ้านของเด็กคนกลาง เพื่อทราบอายุ
   - บันทึกการเก็บของกลาง และสภาพของกลาง
   - หากเด็กได้รับบาดเจ็บ ต้องส่งแพทย์ตรวจและเก็บผลตรวจไว้
5. สถานที่เกิดเหตุและของกลาง ควรถ่ายรูปไว้ และพิจารณาส่งตรวจพิสูจน์เพิ่มเติม
6. การดำเนินการทุกขั้นตอน ต้องบันทึกไว้เป็นหลักฐาน โดยเฉพาะที่มาของพยานหลักฐาน
7. หากผู้ต้องหารับสารภาพ ต้องพาไปชี้ที่เกิดเหตุ ทำแผนประทุษกรรม และสอบสวนเพิ่มเติม
8. พยานที่ไม่เคยเห็นผู้ต้องหามาก่อน
   - ต้องจัดให้มีการชี้ตัว
   - หากเคยรู้จัก → ให้ดูตัวก่อนพาไปชี้ที่เกิดเหตุ
