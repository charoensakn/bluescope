# การสอบสวนคดีลักทรัพย์ในเคหสถานเวลากลางคืน โดยผ่านสิ่งกีดกั้นสำหรับคุ้มครองบุคคลหรือทรัพย์ หรือรับของโจร

1. ต้องรีบไปตรวจสถานที่เกิดเหตุหาร่องรอยพยานหลักฐาน ณ ที่เกิดเหตุโดยละเอียด เช่น
   1. สถานที่เก็บทรัพย์สินที่สูญหายหรือถูกประทุษร้าย ว่าสิ่งของหรือทรัพย์สินแต่ละอย่างเก็บไว้ที่ใด เช่น ในเซฟ ตู้ โต๊ะ ชั้นวางของ ที่ซุกซ่อน เพื่อความปลอดภัย ในห้องนอน ห้องนั่งเล่น ห้องรับแขก ห้องเก็บของ
   2. ร่องรอยงัดแงะ ทุบ เจาะ ไขกุญแจ ตัดสิ่งที่เป็นไม้ เหล็ก กระจก
   3. เครื่องมือเครื่องใช้ของส่วนตัวของคนร้ายที่ลืมทิ้งไว้หรือทำตกไว้ ณ ที่เกิดเหตุ
   4. รอยเท้า ลายนิ้วมือแฝง รอยฟัน เศษไม้ กระจก รอยสว่าน ชิ้นส่วนต่าง ๆ ที่ตกเรี่ยราดอยู่บริเวณที่เกิดเหตุ รอยเลือด (คราบโลหิตในกรณีคนร้ายได้รับบาดเจ็บ) และอื่น ๆ
2. เคหสถานนั้นหมายถึงที่อยู่อาศัย ต้องตรวจลักษณะของอาคารสถานที่ เริ่มตั้งแต่ทางเข้าออก รั้ว กำแพงบ้าน ตัวบ้าน เป็นตึกหรือไม้ ชั้นเดียว สองชั้น สามชั้น มีประตู หน้าต่าง กันสาด ลูกกรงเหล็ก ห้องนอน ห้องน้ำ ที่เก็บทรัพย์สินสิ่งของที่ถูกประทุษร้าย
3. ต้องถ่ายรูปและทำแผนที่สังเขปไว้ด้วย
4. การลักทรัพย์ในเวลากลางคืนนั้น ต้องสอบ
   1. พระอาทิตย์ขึ้นหรือยัง พระอาทิตย์ตกหรือยัง เวลาเท่าไร ถ้าไม่ชัด ต้องสอบเจ้าหน้าที่ของกรมอุตุนิยม สำนักนายกรัฐมนตรีประกอบ
   2. ในขณะเกิดเหตุ ถ้าคนร้ายใช้เรือเป็นพาหนะ และสถานที่เกิดเหตุอยู่ริมแม่น้ำ ลำคลอง ต้องสอบสวนให้ได้ความว่า ความลึกของน้ำ เรือสามารถแล่นเข้าออกบริเวณนั้นได้หรือไม่ ต้องสอบสวนเกี่ยวกับน้ำขึ้น น้ำลงไว้ด้วย ทั้งนี้ต้องสอบสวนเจ้าหน้าที่ของกรมอุตุนิยมเช่นเดียวกับข้อ 4.1 ประกอบ
5. สอบสวนเจ้าของทรัพย์สินที่เสียหายหรือถูกประทุษร้ายในรายละเอียดให้มากที่สุด เช่น ทรัพย์ที่ถูกประทุษร้ายหรือสูญหายนั้น มีตำหนิรูปพรรณอย่างไร มีหลักฐานอย่างไร เช่น รูปถ่าย ตั๋ว พิมพ์รูปพรรณ ใบเสร็จรับเงิน ฯลฯ เป็นต้น ราคาทรัพย์แต่ละอย่างเท่าไร รวมเป็นทรัพย์สินที่สูญหายหรือถูกประทุษร้ายราคาเท่าไร
6. ถ้าได้ของกลางคืน ต้องยึดไว้ตรวจสอบความถูกต้อง ให้เจ้าของหรือผู้ครอบครอง หรือผู้เสียหายยืนยัน
7. ถ้าจับผู้ต้องหาได้พร้อมด้วยของกลาง ต้องสอบสวนถึงการได้มาว่ามาจากไหน เมื่อไร มีหลักฐานอย่างไร เถียงกรรมสิทธิ์หรือไม่
8. ถ้าจับยานพาหนะ เครื่องมือ เครื่องใช้ในการกระทำผิดได้ ต้องสอบสวนสถานที่ได้มาอย่างละเอียด และจะต้องยึดไว้เป็นหลักฐานประกอบคดี
