# การสอบสวนลักทรัพย์ที่เป็นพระพุทธรูปหรือวัตถุในทางศาสนาในที่บูชาสาธารณะอันเป็นที่สักการะบูชาของประชาชน

## 1. การสืบสวนต้องดำเนินการดังนี้

1. ตรวจร่องรอยหาพยานหลักฐานสถานที่เกิดเหตุและบริเวณใกล้เคียงอย่างละเอียด ทางเข้า ทางออก วิธีการเข้าออก การทำลาย งัด แงะ ประตูหน้าต่าง สิ่งกีดกั้นสำหรับคุ้มครองทรัพย์ รอยตะไบ รอยเลื่อย หลักฐานต่าง ๆ ที่ตกอยู่บริเวณสถานที่เกิดเหตุ
2. สืบสวนเบื้องต้น ก่อนเกิดเหตุ ขณะเกิดเหตุ และหลังเกิดเหตุในระยะเวลาใกล้เคียง มีผู้ใดเห็นคนร้าย หรือบุคคลที่น่าสงสัยว่าจะเป็นคนร้ายบ้าง ต้องสืบสวนถึง
   - เพศ
   - รูปร่าง
   - ลักษณะท่าทาง
   - การแต่งกาย
   - อายุ
   - ความพิการของร่างกาย
   - จุดเด่นที่มองเห็นได้ชัด
   - สำเนียงการพูด
   - ยานพาหนะที่ใช้
3. จดตำหนิรูปพรรณทรัพย์หายอย่างละเอียด พร้อมกับราคาทรัพย์ที่หาย เพื่อให้เจ้าหน้าที่ตำรวจอื่นช่วยเหลือในการสืบสวนหาพยานหลักฐาน เพื่อนำไปสู่ตัวผู้กระทำผิด

## 2. การสอบสวนต้องรีบดำเนินการดังนี้

1. สอบสวนผู้เสียหายหรือผู้กล่าวหาให้ได้ความว่า
   - พระพุทธรูปที่หายไปเป็นพระพุทธรูปสร้างด้วยอะไร เช่น หล่อ ปั้น ปั๊ม หรือแกะสลัก
   - ศิลปะยุคหรือสมัยใด ปางอะไร
   - มีตำหนิรูปพรรณอย่างไร
   - ขนาด เช่น หน้าตักกว้างเท่าไร สูงเท่าไร (ถ้ายืนสูงเท่าใด)
   - อายุประมาณกี่ปี
   - ได้ลงสมุดคุมหรือทำบัญชีทะเบียนและถ่ายรูปไว้หรือไม่
   - ราคาเท่าไร

   หรือถ้าเป็นวัตถุทางศาสนา
   - เป็นวัตถุอะไร
   - เป็นที่เคารพบูชาในลัทธิหรือศาสนาใด

2. เก็บรักษาไว้ที่ไหน อย่างไร เป็นวัด สำนักสงฆ์อะไร ตั้งอยู่ตำบล อำเภอ จังหวัดใด มีไว้สำหรับเป็นที่สักการะบูชาของประชาชนทั่วไปหรือไม่ หรือเก็บไว้เป็นพิพิธภัณฑ์ เพื่อเป็นสมบัติของชาติหรือของประชาชนส่วนรวมในท้องถิ่นบริเวณสถานที่เกิดเหตุ

3. สถานที่เก็บรักษา
   - มีคนเฝ้าดูแลรักษาหรือไม่
   - มีสิ่งกีดกั้นที่เก็บพระพุทธรูปหรือวัตถุทางศาสนานั้นหรือไม่
   - ทำด้วยอะไร
   - สิ่งกีดกั้นนั้นถูกทำลายตรงไหนหรือไม่
   - ถ้าถูกทำลาย เสียหายมากน้อยเพียงใด
   - และถูกทำลายด้วยอะไร

4. ถ้าจับกุมผู้ต้องหาได้ และผู้ต้องหาให้การรับสารภาพ ต้องสอบสวนให้ละเอียด ตั้งแต่
   - เริ่มคิดหรือวางแผนในการไปลัก โดยมีเจตนาประสงค์ต่อทรัพย์นั้นอย่างไร
   - ไปกับใครบ้าง
   - วัน เดือน ปี เวลาใด
   - ก่อนเกิดเหตุไปพบกับใคร ที่ไหน
   - ใช้พาหนะอะไร ของใคร มีหลักฐานอย่างไร
   - เครื่องมือเครื่องใช้ในการงัดแงะ เจาะ ตัด มีอะไรบ้าง
   - เข้าทำการลักทางไหน ทำอย่างไรที่เข้าไปในสถานที่เก็บรักษาพระพุทธรูปได้
   - เมื่อพบพระพุทธรูปแล้วทำอย่างไร
   - มีใครช่วยยก หาม ออกทางไหน
   - แล้วนำไปไหน
   - ของกลางที่ถูกจับได้เป็นของผู้ต้องหาหรือไม่
   - ให้ผู้เสียหายหรือผู้กล่าวหาดูของกลางว่าเป็นพระพุทธรูปหรือวัตถุในทางศาสนาหรือไม่
   - ให้ผู้เสียหายหรือผู้กล่าวหาดูของกลางว่าเป็นพระพุทธรูปหรือวัตถุในทางศาสนาที่หายไป ตามวัน เวลา และสถานที่เกิดเหตุหรือไม่

5. พระพุทธรูปหรือวัตถุในทางศาสนา ต้องเรียกชื่อให้ถูกต้องตามหลักนิยม เช่น พระพุทธรูปศิลปะสุโขทัยยุคต้น ปางมารวิชัย หรือปางสะดุ้งมาร หล่อด้วยทองสัมฤทธิ์ มีอายุประมาณ 800 ปี ถือได้ว่าเป็นวัตถุโบราณ เป็นต้น ควรจะให้ผู้รู้ หรือผู้ชำนาญ หรือผู้เชี่ยวชาญ ตรวจสอบยืนยันให้ความเห็น แล้วสอบสวนปากคำประกอบสำนวนการสอบสวนไว้
