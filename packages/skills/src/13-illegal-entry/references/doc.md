# การสอบสวนคดีความผิดเข้ามาในราชอาณาจักรไม่รับอนุญาต

1. การสอบสวนผู้กล่าวหาและพยานควรเน้นถึง
   1. เหตุที่รู้ว่าผู้ต้องหาเป็นคนต่างด้าวหลบหนีเข้าเมือง
   2. ผลการสืบสวนได้ความว่าอย่างไรบ้างมีบันทึกการสืบสวนประกอบ
   3. ผู้ที่สมควรที่จะต้องสอบสวนเป็นพยาน เช่นบุคคลที่อยู่ใกล้กันที่รู้เห็นว่าเป็นคนต่างด้าวหลบหนีเข้าเมืองมาอยู่กับใครที่ไหน มาทำงานอะไร และมาอยู่นานเท่าไร โดยเฉพาะกำนัน ผู้ใหญ่บ้าน สารวัตร กำนัน ผู้ช่วยผู้ใหญ่บ้าน ตลอดจนแพทย์ประจำตำบล หรือบุคคลที่เชื่อถือได้
2. เอกสารต่าง ๆ ที่เกี่ยวข้อง เช่น ใบสำคัญประจำตัว ทะเบียนบ้าน ระเบียบข้อบังคับ คำสั่งของทางราชการ ประกาศของกระทรวงมหาดไทยที่กำหนดช่องทาง ด่านตรวจคนเข้าเมือง เขต ท่า กำหนดเวลาต้องถ่ายภาพ หรือสำเนาติดสำนวนการสอบสวนเพราะเป็นข้อเท็จจริง
3. การสอบสวนผู้ต้องหาควรเน้นถึง
   1. ผู้ต้องหามีเชื้อชาติ สัญชาติอะไร เกิดที่ไหน เช่น หมู่บ้าน ตำบล อำเภอ จังหวัด ประเทศ และ วัน เดือน ปี ใด
   2. เข้ามาในราชอาณาจักรกับใคร เมื่อ วัน เดือน ปี ใด
   3. เข้ามาทางไหน เช่น หมู่บ้าน ตำบล อำเภอ จังหวัดใด หรือช่องทาง ด่านตรวจคนเข้าเมือง เขต ท่าใด และเข้ามาโดยวิธีใด (เดิน, ว่ายน้ำ, นั่งเรือ, รถยนต์, เกวียน, ยานพาหนะอื่น ๆ )
   4. พยานที่รู้เห็นมีใครบ้าง ไปพักกับใครที่ไหน ใครเป็นผู้ให้ที่พักอาศัย และให้ความอุปการะ ได้แจ้งให้เจ้าพนักงานเจ้าหน้าที่หรือกำนัน ผู้ใหญ่บ้าน ทราบหรือไม่ ถ้าแจ้ง ๆ เมื่อไร
   5. ผู้ต้องหามีเอกสารเกี่ยวกับเชื้อชาติ สัญชาติ ติดตัวมาหรือเปล่า ถ้ามีได้สำเนาหรือถ่ายภาพติดสำนวนการสอบสวนไว้
4. คดีคนต่างด้าวเข้ามาในราชอาณาจักรไทยไม่รับอนุญาตนั้น แม้จะอยู่ในราชอาณาจักรเป็นเวลานานก็ไม่ ทำให้คดีขาดอายุความที่จะเป็นเหตุให้ดำเนินคดีกับผู้ต้องหาไม่ได้ เพราะถือว่าเป็นความผิดต่อเนื่อง
5. ต้องจัดทำสำนวนการสอบสวน 3 สำนวนคือ
   1. สำนวนที่ 1 เก็บไว้ที่ทำการของพนักงานสอบสวนเช่น สน. สภ. ตม.
   2. สำนวนที่ 2 ส่งพนักงานอัยการ, ผู้ว่าคดีศาลแขวง
   3. สำนวนที่ 3 ส่ง ตม. ที่ใกล้เคียง เว้นพนักงานสอบสวน ตม. ทำการสอบสวนเอง
6. เมื่อสอบสวนคดีเสร็จสิ้นแล้ว แม้คดีจะอยู่ระหว่างการพิจารณาของพนักงานอัยการ หรือการพิจารณาของศาล พนักงานสอบสวนจะต้องขออายัดตัวคนต่างด้าว ซึ่งเป็นผู้ต้องหานั้นไว้โดยไม่ต้องคำนึงถึงว่าคดีจะถึงที่สุดหรือไม่ เมื่อคดีถึงที่สุดแล้ว ให้นำตัวส่ง ตม. ที่ใกล้เคียง (ตม.จังหวัดหรืออำเภอ) เพื่อพิจารณาดำเนินการส่งตัวออกนอกราชอาณาจักรต่อไป
7. การสอบสวนคดีคนต่างด้าวเข้ามาในราชอาณาจักรโดยไม่รับอนุญาตนี้ พนักงานสอบสวนจะต้องสอบสวนพยานหลักฐานให้ปรากฏให้ชัดว่าผู้ต้องหาได้หลบหนีเข้ามาในราชอาณาจักรโดยวิธีใด ยานพาหนะอะไร มาช่องทาง, ด่านตรวจคนเข้าเมือง, เขตท่า, สถานีหรือท้องที่ หมู่บ้าน, ตำบล, อำเภอ, จังหวัดใด, เมื่อวัน, เดือน, ปี เวลาใด
