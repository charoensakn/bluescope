# การสอบสวนความผิดตาม พ.ร.บ.การพนัน

## 1. การสอบสวนผู้กล่าวหาให้ได้ความว่า

1. ก่อนมีการจับกุมได้ทำการสืบสวนด้วยตนเอง หรือมีสายลับนำความมาแจ้งเพื่อนำจับขอรับเงินรางวัล ถ้ามีผลการสืบสวนหรือที่มีผู้นำความมาแจ้งนั้นได้ความว่าอย่างไร
2. มีผู้เล่นหรือผู้เข้าเล่น หรือเข้าพนันกี่คน
3. เล่นการพนันชนิดอะไร เป็นการพนันตามที่ระบุในบัญชี ก. หรือบัญชี ข. ท้าย พ.ร.บ.การพนัน พ.ศ.2478 หรือการเล่นซึ่งมีลักษณะคล้ายกัน หรือไม่ หรือเรียกชื่อตามภาษาพื้นเมืองว่าอย่างไร
4. พนันเอาทรัพย์สินกัน หรือมีการรับกินรับใช้กันหรือไม่
5. สถานที่เล่นการพนัน เป็นห้องเล่น หรือตั้งวงเล่นในที่สาธารณสถาน หรือสถานที่จัดให้มีงานนักขัตฤกษ์ หรืองานขนบธรรมเนียมประเพณี
6. อุปกรณ์ในการเล่นมีอะไรบ้าง
7. ได้ร่วมกับใครบ้างที่ไปทำการสืบสวนจับกุม
8. จับกุมได้ผู้เล่นการพนันกี่คน ผู้ที่อยู่ในวงการพนันมีกี่คน ได้ของกลางมีอะไรบ้าง
9. ผู้ต้องหาให้การอย่างไร
10. เหตุเกิดที่ไหน เมื่อใด

## 2. การสอบสวนผู้ต้องหาให้ได้ความว่า

1. ให้การรับสารภาพ ภาคเสธ หรือปฏิเสธ
2. เคยต้องหาต้องโทษในคดีอาญามาก่อนหรือไม่ โดยเฉพาะเคยถูกศาลพิพากษาเรื่องการพนันมาก่อนหรือไม่
3. เล่นการพนันอะไรบ้าง ใครเป็นผู้จัดให้มีการเล่น และใครเป็นเจ้ามือจัดให้มีการเล่น
4. อุปกรณ์การเล่นมีอะไรบ้าง ได้เงินหรือทรัพย์สินอะไร เท่าไร หรือเสียเงินหรือทรัพย์สินอะไร เท่าไรในการเล่นครั้งนี้
5. ใครนั่งตรงไหน ทางซ้ายทางขวาของใคร เริ่มเล่นตั้งแต่วัน เดือน ปี เวลาใด และเล่นที่ไหน
6. อุปกรณ์ในการเล่น เช่น เสื่อ ตะเกียง (ถ้ามี) เป็นอุปกรณ์ในการเล่นหรือไม่ ถ้าไม่มีตะเกียงเพื่อให้เกิดแสงสว่าง เสื่อสำหรับปูนั่ง จะเล่นได้หรือไม่ (ถ้าเหตุเกิดเวลากลางคืน)

## 3. การสอบสวนพยาน

ต้องสอบให้ได้รายละเอียดสอดคล้องต้องกันกับผู้กล่าวหา ในกรณีร่วมกับพวกจับกุมผู้ต้องหานั้น ต้องให้ได้ความว่า ผู้ต้องหาเล่นการพนันชนิดอะไร และจะต้องพนันเอาทรัพย์สินกัน และได้มีการรับกินรับใช้ ของกลางที่จับได้ต้องเป็นอุปกรณ์หรือเครื่องมือเครื่องใช้ในการเล่นการพนันนั้น
