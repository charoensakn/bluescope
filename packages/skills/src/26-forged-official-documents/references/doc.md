# การสอบสวนความผิดปลอมแปลงเอกสารราชการและใช้เอกสารราชการปลอม

## ความหมายของ “ปลอม”

“ปลอม” หมายถึง การทำสิ่งเทียมขึ้นโดยเจตนาให้ผู้อื่นหลงเชื่อว่าเป็นของแท้ รวมถึงการ

- แก้ไข
- ตัดทอนข้อความ
- ประทับตราปลอม
- ลงลายมือชื่อปลอม

## องค์ประกอบความผิดในการปลอมเอกสาร

1. เอกสารปลอมต้อง **ไม่ใช่ของจริง**
2. ต้องมี **การกระทำลงไปในเอกสาร**
3. การกระทำนั้น **อาจก่อให้เกิดความเสียหายแก่ผู้อื่นได้**
4. ต้องมี **เจตนาทุจริต**

## ขั้นตอนการสอบสวน

### 1. ตรวจสอบประเภทของเอกสารของกลาง

เอกสารที่ถูกปลอมอาจเป็น

1. เอกสารสิทธิที่เป็นเอกสารราชการ
2. พินัยกรรม
3. ใบหุ้น ใบหุ้นกู้ หรือใบสำคัญที่เกี่ยวข้อง
4. ตั๋วเงิน

### 2. สอบสวนลักษณะการปลอมแปลง

ต้องให้ปรากฏชัดว่าเป็นการกระทำอย่างใด เช่น

1. ทำเอกสารปลอมขึ้นทั้งฉบับ
2. ทำปลอมเฉพาะบางส่วน
3. ตัดทอนหรือแก้ไขข้อความในเอกสารจริง
4. ประทับตราปลอม หรือลงลายมือชื่อปลอม
5. กรอกข้อความลงในกระดาษที่มีลายเซ็นผู้อื่นโดยไม่ได้รับอนุญาต
6. กระทำเพื่อให้ผู้อื่นหลงเชื่อว่าเป็นเอกสารจริง
7. นำเอกสารปลอมไปใช้ในกิจการที่อาจทำให้ผู้อื่นเสียหาย

### 3. สอบสวนพยานผู้เห็นเหตุการณ์

ต้องสอบสวนว่า

- ผู้ต้องหากระทำที่ไหน
- เมื่อไร
- ต่อหน้าใคร
- มีใครรู้เห็นในขณะทำการปลอม

### 4. ตรวจสอบเอกสารของกลาง

ต้องสอบสวนว่าเอกสารนั้น

- เป็นของใคร
- หน่วยงานใดเป็นผู้ออก
- เป็นเอกสารจริงหรือไม่
- มีการแก้ไขเพิ่มเติมตรงจุดใด
- เอกสารจริงออกให้ใคร ที่ไหน เมื่อไร

### 5. สอบสวนผู้ต้องหาในกรณีรับจ้างทำเอกสารปลอม

ต้องสอบสวนรายละเอียดดังนี้

1. เอกสารปลอมทำโดยผู้ต้องหาเองหรือผู้อื่น
2. หากทำเอง ต้องระบุสถานที่ เวลา ผู้ที่รู้เห็น และวัตถุประสงค์
3. เอกสารของกลางที่ยึดได้เป็นของที่ผู้ต้องหาทำขึ้นหรือไม่
4. หากผู้อื่นทำให้ ต้องสอบว่าเป็นใคร อยู่ที่ไหน รู้จักกันอย่างไร และค่าจ้างเท่าไร
5. ผู้ต้องหาทราบหรือไม่ว่าเอกสารนั้นเป็นเอกสารปลอม
6. ผู้ต้องหานำเอกสารปลอมไปใช้ที่ไหน ใช้เพื่ออะไร และถูกจับกุมที่ใด เมื่อไร

### 6. การตรวจพิสูจน์ลายมือชื่อและลายมือหนังสือ

กรณีปลอมลายเซ็นหรือลายมือชื่อ ต้องดำเนินการดังนี้

- สืบหาลายเซ็น ลายมือชื่อ หรือตัวหนังสือตัวอย่างของผู้ต้องหา
- ให้ผู้ต้องหาเขียนตัวอย่างต่อหน้าพนักงานสอบสวน
- ต้องมีตัวอย่างจำนวนเพียงพอ
- ส่งเอกสารไปตรวจพิสูจน์โดยผู้เชี่ยวชาญของกองพิสูจน์หลักฐาน
- ปฏิบัติตาม ป.เกี่ยวกับคดี ลักษณะ 15 บทที่ 5
- แนบผลการตรวจพิสูจน์ไว้ในสำนวน
