# การสอบสวนคดีแต่งเครื่องแบบตำรวจโดยไม่มีสิทธิ์ ไม่มีสิทธิ์ใช้ยศ ตำแหน่งเครื่องราชอิสริยาภรณ์ หรือสิ่งที่หมายถึงเครื่องราชอิสริยาภรณ์

## ศึกษาเรื่องเครื่องแบบตำรวจ

1. เครื่องแบบตำรวจนั้นย่อมประกอบด้วย หมวก เสื้อ กางเกง รองเท้า เครื่องหมายยศ จำพวก สังกัด และอื่น ๆ กับเครื่องประกอบต่าง ๆ
2. ลักษณะ ชนิด และประเภท ของเครื่องแบบตำรวจนั้น จะแต่งอย่างไร เมื่อไร มีเงื่อนไขในการแต่งอย่างไร ต้องเป็นไปตามข้อกำหนดในกฎกระทรวง
3. ต้องศึกษา พ.ร.บ. เครื่องแบบตำรวจ พ.ศ. 2477 (ฉบับที่ 2)
4. การแต่งเครื่องแบบตำรวจต้องปฏิบัติตาม ป. ไม่เกี่ยวกับคดีลักษณะที่ 16 บทที่ 1 ถึง บทที่ 23 ระเบียบกรมตำรวจว่าด้วย "การแต่งเครื่องแบบ" โดยตลอด

## การสอบสวน

### 1. การสอบสวนผู้กล่าวหาควรสอบสวนในประเด็น

1. เห็นผู้ต้องหาแต่งเครื่องแบบตำรวจ ลักษณะ ชนิด และประเภทใด ที่ไหน เมื่อไร
2. แต่งเครื่องแบบชนิดนั้นถูกต้องครบถ้วนหรือไม่ เช่น หมวก เสื้อ กางเกง เข็มขัด รองเท้า ถุงเท้า เครื่องหมายยศ จำพวก สังกัด
3. ประดับแพรแถบเครื่องหมายเครื่องราชอิสริยาภรณ์ หรือสิ่งที่หมายถึงเครื่องราชอิสริยาภรณ์ถูกต้องหรือไม่
4. แต่งเครื่องแบบตำรวจเพื่อกระทำความผิดทางอาญาหรือไม่
5. เมื่อพบครั้งแรกเชื่อหรือไม่ว่าผู้ที่แต่งเครื่องแบบตำรวจนั้นเป็นตำรวจที่แท้จริง

### 2. การสอบสวนผู้ต้องหา

1. มีสิทธิ์แต่งเครื่องแบบตำรวจโดยใช้เครื่องหมายยศ จำพวก สังกัด ประดับเครื่องหมายอื่น และประดับแพรแถบเครื่องราชอิสริยาภรณ์หรือไม่
2. เครื่องแบบและส่วนประกอบของเครื่องแบบเช่น เสื้อ กางเกง เข็มขัด เครื่องหมายยศ สังกัด จำพวกแพรแถบเครื่องราชอิสริยาภรณ์ไปได้มาจากไหน
3. แต่งเครื่องแบบตำรวจทำไม เพื่อประสงค์อะไร
4. รับราชการเป็นตำรวจจริงหรือไม่ ถ้าจริงรับราชการอยู่ที่ไหนตำแหน่งอะไร มีหน้าที่อย่างไร ถ้าเคยรับราชการเป็นตำรวจมาก่อนเคยรับราชการตำรวจมียศตำแหน่งใด อยู่ที่ไหน สาเหตุที่ออกจากราชการเพราะอะไร เช่นลาออก ถูกให้ออก ปลดออก ไล่ออก หรืออยู่ระหว่างพักราชการ
5. ไปแต่งเครื่องแบบตำรวจที่ไหน มาจากไหน และจะไปไหน ไปทำอะไร

### 3. การสอบสวนพยาน

1. เห็นผู้ต้องหาแต่งเครื่องแบบตำรวจ ลักษณะชนิด ประเภทใดที่ไหน
2. ผู้ต้องหาทำอะไรกับใครที่ไหน
3. พยานเชื่อหรือไม่ว่าเป็นเจ้าหน้าที่ตำรวจที่แท้จริง
4. พยานเห็นคนเดียวหรือหลายคนมีใครบ้าง
5. ลักษณะท่าทางและพฤติการณ์ของผู้ต้องหาเป็นอย่างไรบ้าง
6. พยานไปแจ้งเจ้าหน้าที่ฝ่ายปกครองหรือตำรวจหรือบุคคลอื่นทราบหรือไม่
