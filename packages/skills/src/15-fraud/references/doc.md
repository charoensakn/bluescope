# ประเด็นคำถามคดีฉ้อโกง

## 1. การถามผู้เสียหาย

1. รู้จักกับผู้ต้องหาและพวกมานานเท่าใด ที่ไหน ผู้ต้องหาทำอาชีพอะไร
2. เมื่อวันเวลาใด ผู้ต้องหามาหาที่ไหน ตำบลอะไร
3. ผู้ต้องหาใช้อุบายหลอกลวงอย่างไร โดยละเอียด
4. ได้มอบทรัพย์สินอะไร รวมราคาเท่าใด ให้กับผู้ต้องหาไปเมื่อวันเวลาใด ที่ไหน หรือเมื่อวันเวลาใด ที่ไหน ได้ทำ ถอนหรือทำลายเอกสารสิทธิ์ให้หรือไม่ เรื่องอะไร อย่างใด
5. มีใครรู้เห็นในการมอบทรัพย์ ทำถอน หรือทำลายเอกสารสิทธิ์ด้วย
6. ได้พบผู้ต้องหาอีกหรือไม่ เมื่อใด ที่ไหน ผู้ต้องหาขอปฏิบัติตามที่หลอกลวงนั้นหรือไม่ หรือขอผัดผ่อนอย่างไร
7. ทราบเรื่องถูกผู้ต้องหาฉ้อโกงเมื่อวันเวลาใด
8. แจ้งความร้องทุกข์ให้พนักงานสอบสวนดำเนินคดีเมื่อวันเวลาใด
9. เคยมีสาเหตุโกรธเคืองกับผู้ต้องหามาก่อนหรือไม่ ถ้ามีเมื่อใด เรื่องอะไร
10. ของทราบชื่อ อายุ และตำหนิรูปพรรณผู้ต้องหาทุกคน โดยละเอียด (ผู้ต้องหาหนี)

## 2. การถามผู้ต้องหารับสารภาพ

1. รู้จักผู้เสียหายนานเท่าใด เคยติดต่อกันอย่างไร
2. เมื่อวันเวลาใด ที่ไหน ได้หลอกลวงผู้เสียหายว่าอย่างไร โดยละเอียด มีใครร่วมรู้เห็นอยู่ด้วย
3. เมื่อวันเวลาใด ที่ไหน ได้ทรัพย์สิ่งใดไปจากผู้เสียหาย ราคาเท่าใดหรือผู้เสียหายทำ ถอนหรือทำลายเอกสารสิทธิ์อะไรให้ โดยมุ่งต่อผลอย่างไร เกิดประโยชน์แก่ตนหรือผู้อื่นอย่างไรบ้าง คิดเป็นเงินเท่าใด

## 3. การถามผู้ต้องหาให้การปฏิเสธ

1. รู้จักหรือเคยติดต่อผู้เสียหายมาก่อนหรือไม่ อย่างไร โดยละเอียด
2. ตามวันเวลาเกิดเหตุ ทำอะไร อยู่ที่ไหน กับใคร ให้เล่ารายละเอียด
3. เคยมายังที่เกิดเหตุบ้างหรือไม่ เมื่อใด เรื่องอะไร พบกับใครบ้าง
4. เคยได้รับทรัพย์หรือให้ผู้เสียหายทำ ถอน หรือทำลายเอกสารสิทธิ์อะไรบ้างหรือไม่ ถ้าเคยให้เล่ารายละเอียดไว้
5. เคยมีสาเหตุโกรธเคืองกับผู้เสียหายหรือไม่ เมื่อใด เรื่องอะไร

## 4. การถามพยาน

1. รู้จักผู้เสียหาย ผู้ต้องหาหรือไม่ นานเท่าใด รู้จักได้อย่างไร
2. ตามวันเวลาเกิดเหตุมาจากไหน พบกับใครที่ไหน รู้เห็นเหตุการณ์อย่างไร โดยละเอียด มีใครร่วมรู้เห็นอีก
3. ผู้ต้องหาได้ทรัพย์อะไรไปจากผู้เสียหาย ราคาเท่าใด หรือผู้เสียหายได้ทำ ถอน หรือทำลายเอกสารสิทธิ์อะไร เมื่อวันเวลาใด ที่ไหน มีใครรู้เห็นหรือไม่
4. หลังจากนั้นได้พบผู้เสียหาย ผู้ต้องหาอีกหรือไม่ รู้เห็นอะไรเพิ่มเติมอีก
5. เคยมีสาเหตุโกรธเคืองกับผู้ต้องหามาก่อนหรือไม่.
