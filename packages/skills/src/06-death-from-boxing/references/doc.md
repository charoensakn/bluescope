# การสอบสวนคดีการตายเกิดจากการชกมวย

## 1. นายสนามมวยหรือผู้จัดให้มีการชกมวย — ประเด็นที่ต้องสอบสวน

1. วัตถุประสงค์ในการจัดให้มีการชกมวย
2. ประเภทการชก — มวยสากล มวยไทย หรือมวยปล้ำ
3. สถานที่และวันเวลาที่จัดการแข่งขัน
4. ได้รับอนุญาตจากเจ้าพนักงานหรือไม่
    - กรุงเทพมหานคร: นายตำรวจชั้นสัญญาบัตรตำแหน่งสารวัตรขึ้นไป
    - จังหวัดอื่น: นายอำเภอหรือปลัดอำเภอหัวหน้ากิ่งอำเภอ
    - ต้องเสนอผ่านผู้บังคับบัญชาและได้รับอนุมัติจากผู้ว่าราชการจังหวัดหรือผู้บัญชาการตำรวจนครบาล
5. มีเวทีและอุปกรณ์พร้อมหรือไม่ เช่น พื้นเวที เชือกกั้น ระฆัง นวม กระจับ ผ้าพันมือ
6. มีแพทย์ประจำสนามหรือไม่
7. มีกรรมการห้ามมวย กรรมการตีระฆัง และกรรมการให้คะแนนหรือไม่
8. มีเจ้าหน้าที่รักษาความสงบเรียบร้อยหรือไม่
9. กติกาการชกที่ใช้
10. สาเหตุการตายของนักมวยเท่าที่เห็น

## 2. แพทย์ประจำสนาม — ประเด็นที่ต้องสอบสวน

1. วุฒิการศึกษา ทำงานที่ไหน จดทะเบียนประกอบโรคศิลป์หรือไม่
2. ก่อนชกได้ตรวจร่างกายนักมวยหรือไม่ สุขภาพของนักมวยทั้งสองเป็นอย่างไร
3. ระหว่างชกได้ดูการชกหรือไม่ เห็นเหตุการณ์อย่างไร ได้ช่วยเหลืออย่างไร
4. เหตุที่นักมวยตายเกิดจากอะไร

## 3. กรรมการห้ามมวยบนเวที — ประเด็นที่ต้องสอบสวน

1. ความรู้เกี่ยวกับการชกมวย เช่น วุฒิพลศึกษา ปวช./ปวส./ปริญญาตรี
2. ประสบการณ์ห้ามมวยบนเวที — กี่ปี
3. ก่อนชกได้ตรวจเครื่องแต่งกายนักมวยหรือไม่ เช่น นวม กระจับ เข็มขัด แองเกิล
4. ระหว่างชก ใครได้เปรียบ–เสียเปรียบ ชกผิดกติกาหรือไม่ ตักเตือนกี่ครั้ง
5. นักมวยที่ตายถูกชกตรงไหน และกรรมการได้ช่วยเหลืออย่างไร

## 4. กรรมการเปรียบนักมวยหรือผู้จัดคู่มวย — ประเด็นที่ต้องสอบสวน

1. รูปร่างลักษณะของนักมวยทั้งคู่
2. น้ำหนักเท่ากันหรือแตกต่างกันเท่าใด
3. เคยได้ยินชื่อเสียงหรือประวัติของนักมวยทั้งคู่หรือไม่
4. ขณะเกิดเหตุได้เห็นเหตุการณ์บนเวทีหรือไม่ เห็นอย่างไร

## 5. พี่เลี้ยงหรือผู้ให้น้ำ — ประเด็นที่ต้องสอบสวน

1. ประวัติการชกของนักมวยที่ตาย
2. ระหว่างชกได้ดูหรือไม่ ใครได้เปรียบ–เสียเปรียบ
3. การชกเป็นไปตามกติกาหรือไม่
4. เหตุการณ์ที่เกิดขึ้นเป็นอย่างไร

## 6. การชันสูตรพลิกศพ

- ต้องชันสูตรพลิกศพตาม ป.วิอาญา มาตรา 150
- เพราะเป็นการตายผิดธรรมชาติ

## 7. การพิจารณาหลังสอบสวนพยานหลักฐาน

1. พิจารณาข้อเท็จจริงที่เกิดขึ้น
2. หากจัดการแข่งขันโดยไม่รับอนุญาต ต้องดำเนินคดีกับผู้จัดหรือเจ้ามือพนัน
3. หากได้รับอนุญาตถูกต้องและชกตามกติกา
   - นักมวยคู่ชกที่ทำให้อีกฝ่ายตาย **ไม่น่าจะมีเจตนาฆ่า**
   - เป็นเพียงการมุ่งหวังชัยชนะตามกติกากีฬา
4. ไม่ต้องจับกุมนักมวยคู่ชก เว้นแต่นักมวยมาปรากฏตัวต่อพนักงานสอบสวน
   - ให้แจ้งข้อหาตาม ป.วิอาญา มาตรา 134
   - แล้วปล่อยตัวโดยประกันหรือไม่ประกัน
   - ทำความเห็น “สั่งไม่ฟ้อง” และส่งสำนวนให้อัยการโดยไม่ชักช้า
