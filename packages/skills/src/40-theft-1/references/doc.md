# ประเด็นคำถามคดีลักทรัพย์

ต้องตรวจและทำแผนที่สถานที่เกิดเหตุด้วยทุกครั้ง

## 1. การถามผู้เสียหายหรือผู้กล่าวหา

1. เห็นทรัพย์ที่หายครั้งสุดท้ายเมื่อวันเวลาใด ทรัพย์เหล่านี้อยู่ไหน (บอกแขวงหรือตำบล เขตหรืออำเภอ จังหวัด)
2. พบว่าทรัพย์หายไปวันเวลาใด พบเองหรือใครเป็นคนพบเห็นก่อน
3. มีทรัพย์อะไรบ้างที่หาย บอกตำหนิรูปพรรณ และตำหนิพิเศษ ถ้าเป็นทรัพย์มีหมายเลขทะเบียน (ขอใบซื้อขาย ทะเบียน หากเป็นรถหรือปืนที่มีทะเบียนกำกับ ให้สำเนาติดสำนวน แจ้งอายัดการโอนตามระเบียบ) ทรัพย์สิ่งใด ราคาเท่าใด ซื้อมานานเท่าใด จากใคร รวมราคาทั้งหมดเท่าใด
4. คนร้ายเข้าทางไหน ออกทางใด มีร่องรอยงัดแงะผิดปกติตรงไหน
5. ถ้าเห็นตัวคนร้าย ให้บอกตำหนิรูปพรรณ อายุ การแต่งกายโดยละเอียด ถ้าไม่เห็นตัวคนร้าย สงสัยว่าเป็นใคร เพราะเหตุใด
6. คนร้ายทิ้งร่องรอย พยานวัตถุอะไรไว้บ้าง ตรงไหน
7. ไปแจ้งความเมื่อวันเวลาใด
8. กรณีทรัพย์หายในบ้านหรือสำนักงาน ให้สอบด้วยว่ามีใครอยู่เมื่อผู้กล่าวหาหรือผู้เสียหายเข้านอน หรือออกจากสำนักงาน มีใครอยู่ในบ้านหรือสำนักงานเป็นคนสุดท้าย
9. มีสาเหตุโกรธเคืองกับใครบ้าง หากรู้ตัวคนร้าย ให้สอบด้วยว่าเคยมีสาเหตุโกรธเคืองกันมาก่อนหรือไม่ ด้วยเรื่องอะไรบ้าง

## 2. การถามผู้ต้องหาให้การรับสารภาพหรือภาคเสธ

ให้นำไปชี้สถานที่เกิดเหตุ ทำแผนสถานที่เกิดเหตุ และบันทึกประกอบคำรับหรือภาคเสธนั้นไว้ และให้ผู้ต้องหาลงชื่อไว้เป็นหลักฐาน

1. ลักไปเมื่อใด ที่ใด
2. ทรัพย์ที่ลักมีอะไร ของใคร บอกตำหนิรูปพรรณทรัพย์โดยละเอียด ตั้งอยู่ตรงไหน ทำการลักพาไปอย่างไร โดยใช้อะไรเป็นพาหนะ
3. มีใครร่วมด้วย บอกชื่อ อายุ ตำหนิรูปพรรณ และที่อยู่ สถานที่จะตามไปพบ รวมทั้งที่อยู่หรือญาติของผู้ต้องหาหรือคนร้ายที่หลบหนี
4. นำทรัพย์ที่ลักไปไว้ที่ไหน ขายให้ใคร อยู่ที่ไหน ไปในราคาเท่าใด
5. เคยต้องโทษมาก่อนหรือไม่ ถ้าเคยคดีอะไร พ้นโทษเมื่อใด

## 3. ผู้ต้องหาให้การปฏิเสธ

ควรตรวจค้นบ้านเพื่อหาอาวุธและทรัพย์ที่ถูกลักไปนั้น และสืบสวนประวัติการดำรงชีพของผู้ต้องหา

1. ทำงานอะไร ที่ไหน มีรายได้เท่าใด หรือเป็นคนว่างงาน
2. ตามวันเวลาเกิดเหตุอยู่ที่ไหน กับใคร เมื่อใด และออกไปจากสถานที่นั้นเมื่อใด
3. ก่อนหรือวันเกิดเหตุ เคยได้ไปยังสถานที่เกิดเหตุหรือไม่ ถ้าไป ให้บอกวันเวลาและรายละเอียดไว้ มีใครรู้เห็นเหตุการณ์นั้นบ้าง
4. เคยกระทำผิดหรือไม่ คดีใด ผลคดีเป็นอย่างไร
5. เคยรู้จักหรือมีสาเหตุโกรธเคืองกับผู้กล่าวหาหรือผู้เสียหายหรือไม่ อย่างไร

## 4. การถามพยาน

หากเป็นประจักษ์พยาน ให้ชี้กำหนดจุดที่อยู่ไว้ในแผนที่เกิดเหตุด้วย

1. วันเวลาเกิดเหตุอยู่ตรงไหน ห่างที่เกิดเหตุเท่าใด
2. เห็นเหตุการณ์อย่างไร คนร้ายกี่คน เป็นใครบ้าง บอกตำหนิรูปพรรณแต่ละคน รวมทั้งการแต่งกาย คนไหนทำอะไร โดยละเอียด
3. ลักทรัพย์สิ่งใด บอกตำหนิรูปพรรณ ราคา ของใคร อยู่ที่ไหน ไป
4. เคยเห็นทรัพย์นั้นมาก่อนหรือไม่ เห็นครั้งสุดท้ายเมื่อใด รู้ว่าหายเมื่อใด มีใครบอกอย่างไร
5. ก่อนทรัพย์หาย ใครเป็นคนปิดประตูหน้าต่าง มีใครอยู่ที่นั้นบ้าง ถ้ามี คนเหล่านั้นทำอะไรกัน
6. คนร้ายเข้ามาทางไหน ออกไปทางไหน ใช้ยานพาหนะใดหรือไม่
7. พบร่องรอยหรือหลักฐานที่คนร้ายทิ้งไว้หรือไม่
8. มีสิ่งพิรุธ เช่น มีรถอะไรมาจอด หรือมีใครมาป้วนเปี้ยนอยู่บริเวณที่เกิดเหตุบ้าง ให้บอกรายละเอียดไว้ (เพื่อเป็นทางสืบสวนสอบสวนต่อไป)
