# การสอบสวนคดีทำให้หญิงแท้งลูก ประกอบโรคศิลปะ ประกอบอาชีพพยาบาล และผดุงครรภ์ไม่รับอนุญาต

## 1. การสอบสวนหญิงที่ไม่ยินยอมให้ทำแท้งลูกต้องสอบสวนให้ได้ความว่า

1. ตั้งท้องกับใคร กี่เดือนแล้ว
2. ยินยอมให้ใครเป็นคนทำแท้งให้หรือไม่
3. เมื่อไม่ยินยอม ผู้นั้นทำแท้งให้ได้อย่างไร
4. ทำที่ไหน เมื่อไร
5. มีใครรู้เห็นการกระทำบ้าง
6. ได้ไปรักษาตัวที่ไหน กี่วัน
7. ได้ไปแจ้งความร้องทุกข์ที่ไหน กับใคร เมื่อใด
8. ต้องสอบสวนหญิงคนกลางไว้ ในฐานะผู้เสียหายหรือผู้กล่าวหา

## 2. การสอบสวนหญิงที่แท้งลูกโดยหญิงนั้นยินยอม ต้องสอบสวนให้ได้ความว่า

1. ตั้งท้องกับใคร กี่เดือนแล้ว
2. เคยเจ็บป่วยเป็นโรคอะไรมาก่อนหรือไม่ เคยเข้าโรงพยาบาลรักษาตัวมากี่ครั้งแล้ว เสียค่าใช้จ่ายมากน้อยเพียงใด
3. สถานภาพทางครอบครัวเป็นอย่างไร
4. การตั้งครรภ์เนื่องมาจากถูกข่มขืนกระทำชำเรา อายุไม่เกินสิบห้าปีถูกข่มขืนกระทำชำเรา ถูกผู้อื่นล่อลวงชักพาไปเพื่ออนาจารหรือสำเร็จความใคร่ของผู้อื่นหรือไม่
5. สมัครใจยินยอมให้ผู้ต้องหาทำแท้งลูกให้หรือไม่
6. ผู้ทำแท้งลูก (ผู้ต้องหา) ชื่ออะไร อยู่ที่ไหน รูปร่างลักษณะตำหนิรูปพรรณเป็นอย่างไร เป็นหมอหรือแพทย์ที่ได้รับอนุญาตหรือไม่
7. ได้ชำระเงินค่าทำแท้งลูกให้กับใคร เท่าไร มีหลักฐานอย่างไร
8. การทำแท้งลูก ผู้ทำให้มีกี่คน จำได้หรือไม่ว่าเป็นใครบ้าง
9. ก่อนทำแท้ง ขณะทำแท้ง หลังทำแท้ง ผู้ทำได้ทำอะไรให้บ้าง เช่น ทำความสะอาด ฉีดยา ถ่างมดลูก ขูดมดลูก นัดให้มาตรวจที่ไหน ผู้ทำแท้งทำอะไรให้บ้าง
10. ได้รับบาดเจ็บที่ไหนบ้าง ไปรักษาตัวที่ไหน กี่วัน ปกติทำงานอะไรที่ไหน ประกอบอาชีพอะไร ไปทำงานปกติได้ภายในกี่วัน
11. ในกรณีหญิงยินยอมให้ผู้อื่นทำแท้ง ต้องสอบสวนหญิงนั้นเป็นพยาน อย่าสอบสวนเป็นผู้เสียหาย หรือผู้กล่าวหา เพราะหญิงนั้นมีส่วนร่วมกระทำผิดด้วย
12. ควรสอบสวนเจ้าพนักงานตำรวจผู้สืบสวนจับกุมเป็นผู้กล่าวหา

## 3. การสอบสวนพยานต้องสอบสวนให้ได้ความว่า

1. รู้จักกับผู้ต้องหามาก่อนหรือไม่
2. รู้เห็นการทำแท้งลูกของผู้ต้องหาอย่างไรบ้าง เช่น
   - ก่อนทำแท้งมีการตรวจร่างกายอย่างไรบ้าง
   - ก่อนทำความสะอาดร่างกายทำที่ไหน ทำอย่างไร
   - การฉีดยา
   - การขูดมดลูก
   - การใช้สิ่งของต่างๆ เข้าไปในช่องคลอด
   - การเก็บสิ่งต่างๆ ที่นำออกช่องคลอด
   - เครื่องมือเครื่องใช้มีอะไรบ้าง เก็บไว้ที่ไหน
   - การจ่ายยา การชำระเงิน
   - การนัดคนป่วยให้มาพบ
3. เคยมีสาเหตุโกรธเคืองกับผู้ต้องหา หรือคนอื่นที่ทำงานด้วยกันมาก่อนหรือไม่
4. ใครเป็นผู้นำหรือเชิญพยานมาให้การกับพนักงานสอบสวน

## 4. การสอบสวนผู้ต้องหาให้ได้ความว่า

1. ถ้าผู้ต้องหาให้การรับสารภาพต้องสอบสวนให้ได้ความว่า
   1. มีความรู้ในการทำคลอดทำแท้งมาจากไหน ใครเป็นคนสอนให้
   2. ทำคลอด ทำแท้งลูกที่ไหน มีใครช่วยเหลือบ้าง
   3. ได้รับผลประโยชน์อย่างไร
   4. ใครเป็นผู้สนับสนุนสั่งให้ทำ
   5. เครื่องมือเครื่องใช้ของกลางที่เจ้าหน้าที่ตำรวจยึดได้เป็นของใคร ได้มาจากไหน แต่ละอย่างใช้ทำอะไร ถ้าเป็นยาเป็นยาชนิดใด ใช้เพื่ออะไร ได้มาจากไหน เป็นของผู้ต้องหาหรือไม่
   6. ประกอบโรคศิลปะได้ขึ้นทะเบียนและได้รับใบอนุญาตจากคณะกรรมการควบคุมการประกอบโรคศิลปะหรือไม่
   7. ประกอบวิชาชีพพยาบาลและการผดุงครรภ์ได้ขึ้นทะเบียนและรับใบอนุญาตเป็นผู้ประกอบอาชีพการพยาบาลและการผดุงครรภ์จากสภาการพยาบาลหรือไม่
   8. ที่ให้การรับสารภาพพาพนักงานสอบสวนไปชี้ที่เกิดเหตุได้หรือไม่
   9. เคยต้องโทษต้องหาคดีอาญามาก่อนหรือไม่ ถ้าเคยเคยถูกศาลพิพากษาปรับ หรือจำคุกหรือไม่ ถ้าจำคุกจำคุกกี่เดือน กี่ปี ศาลรอการลงโทษให้หรือไม่อย่างไร
2. ถ้าผู้ต้องหาให้การปฏิเสธต้องสอบสวนให้ได้ความว่า
   1. เคยต้องหาต้องโทษคดีอาญามาก่อนหรือไม่
   2. สถานที่เกิดเหตุเป็นบ้านใคร
   3. ของกลางที่เจ้าหน้าที่ตำรวจยึดได้ เป็นของใคร มีไว้ทำไม ใครเป็นผู้ใช้และใครเป็นผู้รักษา
   4. พยานชี้ตัวผู้ต้องหา เคยรู้จักกันและมีสาเหตุโกรธเคืองกันมาก่อนหรือไม่

## 5. การสอบสวนผู้กล่าวหาหรือพยานจับกุม ต้องสอบสวนให้ได้ความว่า

1. การสืบสวนก่อนมีการจับกุม ได้พยานหลักฐานอะไรบ้าง
2. การจับกุมผู้ต้องหา พบผู้ต้องหา หญิงคนกลางผู้ถูกทำแท้งที่ไหน กำลังทำอะไรอยู่กับใคร
3. ยึดของกลางได้อะไรบ้าง
4. ของกลางที่ยึดได้ประกอบคดีเป็นของใคร ยึดได้ที่ไหน
5. ผู้ต้องหาให้การขณะจับกุมว่าอย่างไร

## 6. หากตรวจยึดของกลางได้ในที่เกิดเหตุควรส่งไปตรวจพิสูจน์ดังนี้

1. หญิงคนกลางมีโลหิตหมู่ใด
2. ก้อนเลือดหรือโลหิตที่ยึดได้ ณ ที่เกิดเหตุเป็นโลหิตของมนุษย์ใช่หรือไม่ หมู่ใด
3. ยาที่ยึดได้เป็นยาชนิดใด ใช้สำหรับทางแพทย์หรือไม่
4. เครื่องมือเครื่องใช้ที่ยึดได้ เป็นเครื่องมือเครื่องใช้สำหรับแพทย์ใช่หรือไม่
5. เส้นผม ขน อาจยึดได้จากที่เกิดเหตุต้องนำไปตรวจพิสูจน์ว่าเป็นเส้นผม ขน ของมนุษย์ และเป็นของหญิงคนกลางใช่หรือไม่
